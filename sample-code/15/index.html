calc()で画像の高さを調整する方法

コード概要

要点

前回、object-fitプロパティを使って、画像（ヒーローイメージ）を画面いっぱいに表示する方法について書きました。今回は、ヘッダーを含めたくない場合の方法です。CSSの calc()関数を使います。ポイントは以下ようです。

  • ヘッダーの高さ（height）を指定する。ここでは、120pxとした。
  • 画像の高さ（height）は、ブラウザ画面の高さから、ヘッダーの高さを引いた値となる。ゆえに「calc(100vh - 120px)」として算出している。

コード

HTML


<header class="header">
  <div class="header__container">
    <h1 class="siteTitle">calc()で高さを調整する方法</h1>
  </div>
</header>
<div class="mainVisual">
  <img class="mainVisual__img" src="mainVisual.jpg">
</div>

CSS


.header__container {
  height: 120px;
}
.mainVisual__img {
  width: 100%;
  max-width: 1920px;
  height: calc(100vh - 120px);
  object-fit: cover;
}

あとがき
