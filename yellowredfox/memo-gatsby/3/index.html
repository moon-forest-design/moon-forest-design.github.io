StaticImageによる画像表示

概要

要点

  1. 右クリックをして管理者権限で Git Bashを起動する。
  2. npmコマンドで、GatsbyJSのプラグインgatsby-plugin-imageをインストールする。gatsby-plugin-imageをインストールする場合は、あわせて gatsby-plugin-sharpとgatsby-source-filesystemもインストールしなければならない。これら 2つは、「peer dependencies（同等依存）」しているプラグイン。
  3. gatsby-config.jsファイルに gatsby-plugin-imageと gatsby-plugin-sharpプラグインに関する設定情報を追加。
  4. src下に画像用のディレクトリ（例：img）を作成し、画像を入れておく。（例：logo-yellowredfox.png）
  5. index.jsなどのファイルに、import 及び表示したい箇所に <StaticImage />を記述。alt属性や src属性を指定する。意味は、img要素と同様。

プラグインgatsby-plugin-imageの導入手順

コマンドライン


// プラグインgatsby-plugin-imageをインストール。
$ npm install gatsby-plugin-image gatsby-plugin-sharp gatsby-source-filesystem

gatsby-config.js


// gatsby-config.jsファイルに設定を追加
module.exports = {
  plugins: [
    "gatsby-plugin-image",
    "gatsby-plugin-sharp",
  ],
};

index.jp


// importする
import { StaticImage } from 'gatsby-plugin-image';

// 表示したい場所に<StaticImage />で指定（以下例）
<StaticImage alt="黄赤狐ロゴ" src="../img/logo-yellowredfox.png" />

編集後記

あとがき

  • プラグインgatsby-plugin-imageを使うことで、パフォーマンスはより良くなるそうだ。また、自動的に最適化されて、レスポンシブ対応になるとのこと。
