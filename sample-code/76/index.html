「class構文によるカウントアップボタン」

コード概要

要点

  • class構文で定義したイベント処理のサンプル カウントアップボタン。
  • JavaScriptの classでは、サブクラスのコンストラクタを定義する際は常に「super」を呼ぶ必要がある。super()と書くことで、親のコンストラクタ（React.Component）を呼び出すことができる。コンストラクタで this（this.stateなど）を使いたい場合は、「super(props);」と書く必要がある。
  • onClick={this.incrementCount} のように ()を末尾に付けずにメソッドを参照する場合は、そのメソッドは bind()しておく必要がある。
  • setState()で、コンポーネントの更新を要求する。ただし、処理効率をより良くするため、即時に更新するとは限らない。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  class CountUpBtn extends React.Component {
    constructor(props) {
      super(props);
      this.state = { count: 1 };
      // bind()は、コールバックで thisを機能させるために必要
      this.incrementCount = this.incrementCount.bind(this);
    }
    incrementCount() {
      // setState()で、コンポーネントの更新を要求する
      this.setState({ count: this.state.count + 1 });
    }
    render() {
      return (
        <div>
          <button onClick={this.incrementCount} className='countBtn'>
            {this.state.count}
          </button>
        </div>
      );
    }
  }

  const render = () => ReactDOM.render(
    <React.StrictMode>
      <CountUpBtn />
    </React.StrictMode>,
    document.getElementById('root')
  );
  render();
</script>
