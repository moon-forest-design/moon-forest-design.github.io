トップへ戻るボタン

コード概要

要点

今回は、トップへ戻るボタンです。仕様は以下ようです。

  • 事前に、トップへ戻るボタンを、CSSで「display: none;」にしておきます。
  • トップへ戻るボタンの表示・非表示を切り替えるラインは、ページの 7割くらいの位置に基準線をもうけています。ページを下方へスクロールして 、基準線を越えると、jQueryでトップへ戻るボタンを表示します。クリックすると、トップの位置へ移動します。
  • トップへ戻るボタンをクリックすることなく、ページを上方へスクロールして基準線に戻った場合は、jQueryでトップへ戻るボタンを非表示にします。

コード

HTML


<a class="back-to-top" href="#">
	<img id="backToTopBtn" class="back-to-top__btn" src="arrow.png">
</a>

CSS


.back-to-top {
	display: block;
	position: fixed;
	right: 0;
	bottom: 0;
	width: 60px;
	height: 60px;
	margin: 3em;
	text-decoration: none;
}
.back-to-top__btn {
	display: none;
	width: 100%;
	height: auto;
	padding: 1em;
	box-sizing: border-box;
	background-color: #241c1c;
	border: 4px solid #fffffc;
	border-radius: 50%;
}

jQuery


$(function() {
	$(window).scroll(function() {
		const $winHeight = $(this).height(); // ウィンドウ画面の高さを取得
		const $scrollTop = $(this).scrollTop(); // スクロールした現在のtopの位置を取得
		const $scrollBottom = $scrollTop + $winHeight; // スクロールした現在のtopの位置にウィンドウ画面の高さを加算してbottomの位置を算出
		const $borderLine = $(document).height() * 0.7; // ページの高さの7割くらいを基準線にする 
		if ( $scrollBottom > $borderLine ) {
			$("#backToTopBtn").fadeIn(); // スクロールして基準線を超えたらトップへ戻るを表示
		} else {
			$("#backToTopBtn").hide(); // スクロールが基準線に達しない場合はトップへ戻るを非表示
		}
	});
});

編集後記

あとがき

スムーズスクロール機能を外してあるので、トップへ戻るボタンを押すとバチっと戻ってしまいます。やはり、スムーズスクロールしたほうがいいですね。
