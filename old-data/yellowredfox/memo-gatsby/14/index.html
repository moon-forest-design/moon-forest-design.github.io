StaticImage画像を中央揃えにする方法

概要

要点

  1. 例えば、index.js内で表示する画像を中央揃えにしたいときの方法。おそらく、もっとスマートな方法が他にあるはずだが、今のところこれしか成功していない。
  2. 方法は以下。
    • <StaticImage />を <figure>要素などで囲う。
    • <figure>要素に対して、flexの justify-contentで centerを指定する。

コード

jsファイル


<figure>
  <StaticImage
    alt="サンプル"
    src="../img/sample.png"
  />
</figure>

cssファイル


figure {
  display: flex;
  justify-content: center;
}

編集後記

あとがき

  • 本来であれば、「objectPosition=""」でうまいこと配置できるはずなのだが、デフォルトは 50% 50%である。試しに centerなどを入れてみても変化はない。そもそも中央に配置されているのだが、親要素が画像の幅と同じ大きさらしく、それでは当然のことながら中央に表示されるはずもない。
  • 既存の親要素はあてにできないようだ。レスポンシブ等うまいこと表示するためにしている処理の邪魔をするのもややこしい話になってしまう。「className=""」や「style=""」も試してはみたが、変化はなかった。
  • 今回は、まるっと <figure>要素で囲み、flexを使って中央揃えにした。スマートな方法ではないが、他に方法がなければこうするしかない。
