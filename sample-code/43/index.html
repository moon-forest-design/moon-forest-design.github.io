Googleマップをセピア調にする方法

コード概要

要点

今回は、Googleマップをセピア調にする方法について書きます。Googleマップを設置したら、あとは CSSで filterプロパティを指定するだけです。Googleマップの場合、値は 40%くらいが良さそうです。

コード

HTML


<div class="googleMap">
  <iframe src="～（省略）～" width="600" height="450" style="border:0;" allowfullscreen="" loading="lazy"></iframe>
</div>

CSS


.googleMap iframe {
  width: 100%;
  filter: sepia(40%);
}

編集後記

あとがき

  • Googleマップを白黒にしたい場合は、「filter: grayscale(100%);」を指定します。
  • filterプロパティは IE非対応です。IEに対応しなければいけない案件であれば他の方法を検討する必要があります。
