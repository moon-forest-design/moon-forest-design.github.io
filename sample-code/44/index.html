CSSで影をつける方法

コード概要

要点

今回は、ドロップシャドウの方法をいろいろと試します。text-shadow、box-shadow、filter（drop-shadow()）など、いろいろとあるようですが、どのような違いがあるのでしょうか。

text-shadow

これまでテキストに影をつけたければ、text-shadowを使ってきました。例えば、「text-shadow: 2px 2px 2px rgba(0, 0, 0, .5);」のように指定することで、次のようにテキスト（文字列）に影をつけることができます。指定するプロパティ値は左から、X軸・Y軸・ぼかし・色です。

テキストテキストテキストテキストテキストテキストテキストテキストテキスト

box-shadow

画像や div要素に影をつけたければ、box-shadowを使います。例えば、「box-shadow: 2px 2px 2px 1px rgba(0, 0, 0, .5);」のように指定することで、次のように画像の縁（ふち）に影をつけることができます。指定するプロパティ値は左から、X軸・Y軸・ぼかし・広がり・色です。

filter（drop-shadow()）：テキスト

filter（drop-shadow()）を使うことで、影をつけることもできます。例えば、「filter: drop-shadow(2px 2px 2px #555);」のように指定することで、次のようにテキストに影をつけることができます。指定するプロパティ値は左から、X軸・Y軸・ぼかし・色です。

テキストテキストテキストテキストテキストテキストテキストテキストテキスト

filter（drop-shadow()）：画像

filter（drop-shadow()）は、テキストだけでなく画像にも影をつけることができます。興味深いのは、png透過画像に影をつけた場合です。例えば、「filter: drop-shadow(2px 2px 2px #555);」のように指定すると透過を無視して、中身（オブジェクト）に対して影がつきます。これは、svg画像でも同様の結果が得られます。

コード

HTML


  <!-- text-shadow -->
  <p class="textShadow">テキストテキストテキストテキストテキストテキストテキストテキストテキスト</p>
  
  <!-- box-shadow -->
  <img class="boxShadow" src="pngSample.png">
  
  <!-- filter（drop-shadow()）：テキスト -->
  <p class="filterDropShadow">テキストテキストテキストテキストテキストテキストテキストテキストテキスト</p>
  
  <!-- filter（drop-shadow()）：画像 -->
  <img class="filterDropShadow" src="pngSample.png">

CSS


/* text-shadow */
.textShadow {
  text-shadow: 2px 2px 2px rgba(0, 0, 0, .5);
}

/* box-shadow */
.boxShadow {
  box-shadow: 2px 2px 2px 1px rgba(0, 0, 0, .5);
}

/* filter（drop-shadow()） */
.filterDropShadow {
  filter: drop-shadow(2px 2px 2px #555);
}

編集後記

あとがき

  • filterのdrop-shadow()は、IEに対応していません。IEに対応させたければ、text-shadowや box-shadowを利用することになりそうです。
