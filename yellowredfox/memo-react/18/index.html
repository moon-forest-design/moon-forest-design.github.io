コード概要

要点

  • event.targetは、「イベントを発生させたオブジェクトへの参照」を行うプロパティ。ここでは、event.target.valueとし、input要素で入力された値を参照することになる。似たような名称のものに、event.currentTargetがある。用途に応じて使い分ける。
  • input要素の value属性は初期状態で空にしておく。入力されたら値を表示するのに使用する。
  • input要素の onChange属性は、入力イベントが発生したら関数setTextDataを起動する役割。
  • 関数setTextData内で 10文字以上入力できないバリデーションを行っている。10文字を越えたら slice()で 10文字までに切り整える。
  • 関数setTextDataが呼び出されるたびに render()で再描画することで、入力した値が value属性値として表示できる。

コード

HTML


<div id="root" class="demoRoot"></div>

jsx


  const style = {
    width: '100%',
    padding: '1em',
    boxSizing: 'border-box',
  };
  let textData = '';
  const setTextData = (event) => {
    textData = event.target.value; // 入力された値を格納
    if (textData.length > 10) {
      textData = textData.slice(0, 10); // 10文字を越えたら10文字までに切り整える
    }
    render(); // 入力されるたびに再描画する
  };
  const InputForm = () => (
    // value属性　入力された値を表示　初期値は空''
    // onChange属性　入力イベントが発生したら関数setTextDataを起動
    <div style={style}>
      <input type="text" value={textData} onChange={setTextData} style={style} />
    </div>
  );
  const render = () => ReactDOM.render(
    <React.StrictMode>
      <InputForm />
    </React.StrictMode>,
    document.getElementById('root')
  );
  render();
