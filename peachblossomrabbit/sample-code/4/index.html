img要素やsvg要素などの画像の下にできる余白を消す方法

コード概要

要点

img要素の下にできる不要な余白を消す方法

身に覚えのないおかしな余白が、img要素の下にできることがあります。marginも paddingも入れていないはずなのに、なぜか少しだけ間（あいだ）が空いている。

「修正しようと思ったけれど、原因が分からなくて無駄に時間を過ごしてしまった」というような経験をしたことが、誰しも一度くらいはあるのではないでしょうか。私はあります。

例えば、次のように、figure要素の中に、img要素を使って入れた画像。やはり、不必要な余白が画像の下にできてしまいます。（余白が分かりやすいように figure要素に背景色をつけてあります）

なぜこんなことが起きるのかと言うと、img要素の vertical-alignの初期値が「baseline」だからなのです。原因が分かれば、改善する方法は簡単です。img要素に「vertical-align: bottom;」を指定してあげるだけで解決します。画像の下の不必要な余白が消えて、ぴたっと無駄なくおさまります。

svg要素の下にできる不要な余白を消す方法

これは、svg要素にも同じことが言えます。やはり、不必要な余白が画像の下にできてしまいます。これじつは、インライン要素の vertical-alignの初期値が「baseline」だから発生するんですよね。

ゆえに、img要素と同様の方法で解決します。svg要素に「vertical-align: bottom;」を指定してあげるだけです。svg要素の下にできた不必要な余白が消えて、ぴたっと無駄なくおさまります。

コード

CSS


img, svg {
 vertical-align: bottom;
}

編集後記

あとがき

  • 画像をブロックレベル要素にしてもかまわなければ、「display: block;」を指定しても良いかもしれません。
