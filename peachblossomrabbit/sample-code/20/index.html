メディアテレコ

コード概要

要点

今回は、「メディアテレコ」です。前回書いたメディアのレイアウトとほぼ同じですが、サムネイル画像とタイトル・テキストを互い違いに配置する違いがあります。仕様は以下ようです。（※：メディアテレコという言葉はありません。私の造語です。）

  • 「flex-direction: row-reverse;」を追加することで、サムネイル画像とタイトル・テキストは逆になります。
  • 疑似クラス:nth-child(even)を使うことで、偶数のときだけ指定されます。
  • 窮屈な印象になるので、各メディア（サムネイル画像とタイトル・テキスト）の下の余白（margin-bottom）の値を増やしています。
チョコレートケーキ

ふわふわのスポンジを4層かさねたチョコレートケーキです。濃厚なクーベルチュールチョコレートを「これでもかっ」というほどに使用。

ブルーベリーレアチーズケーキ

レアチーズをスポンジではさみ、ブルーベリーソースをかけたケーキです。さらに、カシスソースをかけてお召しあがりください。

ストロベリーロールケーキ

相性の良い苺（いちご）の酸味とこだわりの生クリームの甘みをお楽しみいただけるであろう極上のロールケーキです。

CSS


.media {
  display: flex;
  margin-bottom: 5em;
}
.media:nth-child(even) {
  flex-direction: row-reverse;
}
.mediaThumbnail {
  flex: 0 1 30%;
}
.mediaThumbnail__img {
  width: 100%;
  vertical-align: top;
  border-radius: 2px;
}
.mediaContent {
  flex: 1;
  padding: 0 2em;
}
.mediaTitle {
  font-size: 1.8rem;
  font-weight: bold;
  margin-bottom: .5em;
}
.mediaTitle__link {
  text-decoration: none;
  color: inherit;
}
@media screen and (max-width: 767px) { /* スマホ用 */
  .media {
    flex-direction: column;
  }
  .mediaThumbnail__img {
    margin-bottom: 1em;
  }
  .mediaContent {
    padding: 0;
  }
  .mediaTitle {
    text-align: center;
  }
