カード

コード概要

要点

今回は、「カード」と呼ばれているレイアウトについて書きます。サムネイル画像・タイトル・テキストなどを一組のグループとして、そののグループを横並びにしたレイアウトを、ホームページ制作においては「カード」と呼びます。仕様は以下ようです。

  • Flexboxを使って横並びにしています。子要素の配置は space-betweenです。
  • 画像の widthは calc()で算出しています。
  • レスポンシブ対応しています。スマートフォンで見た場合はすべて 1カラム表示になります。

CSS


.cardBox {
  display: flex;
  justify-content: space-between;
  padding: 1.5em 0;
}
.card {
  height: auto;
  margin-bottom: 2em;
  text-align: center;
}
.cardBox--col2 .card {
  width: calc(97% / 2);
}
.cardBox--col3 .card {
  width: calc(97% / 3);
}
.cardBox--col4 .card {
  width: calc(97% / 4);
}
.cardThumbnail__img {
  width: 100%;
  height: auto;
  margin-bottom: .5em;
  vertical-align: top;
  border-radius: 2px;
}
.cardTitle__link {
  text-decoration: none;
  color: inherit;
}
@media screen and (max-width: 767px) { /* スマホ用 */
  .cardBox {
    flex-direction: column;
  }
  .cardBox--col2 .card,
  .cardBox--col3 .card,
  .cardBox--col4 .card {
    width: 100%;
  }
}

編集後記

あとがき

  • シンプルに横並びさせる部分だけのコードです。画像の枚数をカラム数にあわせて用意して良ければ、今回のように簡単に横並びさせることができます。
  • ここでは、タイトルを div要素で囲みましたが、状況に応じてヘディング要素（h3要素など）に変更することでより良いマークアップとなります。
  • 画像の枚数がカラム数を超える場合は、space-betweenの最後の行を左寄せにする方法で書いたように、flex-wrap: wrap;で折り返したり、jQueryで最後の行を調整する必要があります。
