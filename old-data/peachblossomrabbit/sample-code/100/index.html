React入門
「検索できる商品データ表」

コード概要

要点

  • React公式の「React の流儀」で紹介されているコード。チュートリアル形式で「検索できる商品データ表」を制作する内容。以下、仕様。
    • 初期状態ですべての商品データが一覧表示される。
    • 検索入力欄に文字入力すると、該当する商品のみデータ表に残り表示される。
    • 「Only show products in stock」にチェックを入れると、在庫のない商品は非表示になる。
      • 　
  • 制作の流れ。
    • Step 1: UIをコンポーネント分割し、階層構造にする
    • Step 2: propsのみで静的なバージョンを作成する
    • Step 3: 最低限必要な stateを定める
    • Step 4: stateを所有するコンポーネントを決める
    • Step 5: 逆方向のデータフローを追加する

制作の流れメモ

Step 1: UIをコンポーネント分割し、階層構造にする

  • すでに、JSON APIが実装済みで、デザイナーから「検索できる商品データ表」のデザインモックを受け取っている状況。
  • モックを作成したツール（Photoshopなど）内で使用したレイヤ名を、デザイナーから聞く。レイヤ名を参考にして、各コンポーネント名にする。
  • コンポーネント分割は、単一責任の原則（single responsibility principle）を指針とする。以下例。
    • FilterableProductTable： 「検索できる商品データ表」の本体
    • SearchBar： 検索領域
    • ProductTable： 商品データ表
    • ProductCategoryRow： カテゴリ見出し
    • ProductRow： 各商品データ（1行で表示）
  • コンポーネント分割ができたら、見やすいように階層構造で並べる。

Step 2: propsのみで静的なバージョンを作成する

  • まずデータを受け取って UIの描画だけを行うバージョンを作る。ユーザからの操作はできなくていい。
  • とりあえず propsを使う。この段階ではまだ stateを使わないでおく。
  • シンプルなアプリは、通常トップダウンで作った方が楽。大きなプロジェクトは、開発をしながらテストを書き、ボトムアップで進める方がより簡単。

Step 3: 最低限必要な stateを定める

  • DRYの原則（don’t repeat yourself）を指針として、アプリケーションが必要とする最小限の構成で作る。必要なものが出てきたら、そのとき計算すればよい。例えば、配列に格納されているデータをカウントしておく変数は不要。lengthで算出できる。
  • stateを使用するべきか判断する。以下、基準。
    • 親コンポーネントから 受け取るだけで済むデータであれば、stateにする必要はない。
    • 時間が経過しても変化しないデータであれば、stateにする必要はない。
    • コンポーネント内にある他のデータ（propsや state）を使って算出可能なデータであれば、stateにする必要はない。
  • 今回使用するデータは 4つ。stateにするのは 2つ。
    • 元となる商品のリスト
    • ユーザが入力した検索文字列（stateにする）
    • チェックボックスの値（stateにする）
    • フィルタ済みの商品のリスト

Step 4: stateを所有するコンポーネントを決める

  • 「摂氏・華氏の変換ツール（stateのリフトアップ）」で学んだように、どのコンポーネントが stateを所有すべきかを考える必要がある。誤った選択はパフォーマンスを悪くする。以下、考える上でのポイント。
    • stateを使うすべてのコンポーネントを確認する
    • 共通する親コンポーネントを見つける（あるいはさらに上位の祖先コンポーネントでもよい）
    • stateを所有するのにふさわしいコンポーネントが見つからない場合は、stateを保持するためだけの新しいコンポーネントを作り、階層構造上において共通する親コンポーネントの上位に祖先コンポーネントとして配置する
  • 今回の「検索できる商品データ表」の場合は、FilterableProductTableが stateを所有するコンポーネントとしてふさわしいことが分かる。

Step 5: 逆方向のデータフローを追加する

  • 階層構造の下位にある子コンポーネントから、親コンポーネントの stateを更新できる仕組みを作る。
  • 今回の「検索できる商品データ表」の場合であれば、FilterableProductTableは SearchBarにコールバックを渡しておいて、stateを更新したいときに実行してもらう形にする。入力のたびに呼び出される onChangeイベントを利用する。このコールバックが実行された FilterableProductTableは、setState()を呼び出し、その結果としてアプリは更新される。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  // カテゴリ見出しを表示
  class ProductCategoryRow extends React.Component {
    render() {
      const category = this.props.category;
      return (
        <tr>
          <th colSpan="2">
            {category}
          </th>
        </tr>
      );
    }
  }
  // 各商品データを表示
  class ProductRow extends React.Component {
    render() {
      const product = this.props.product;
      const name = product.stocked ?
        product.name :
        <span style={{color: 'red'}}>
          {product.name}
        </span>;

      return (
        <tr>
          <td>{name}</td>
          <td>{product.price}</td>
        </tr>
      );
    }
  }
  // 商品データ表
  class ProductTable extends React.Component {
    render() {
      const filterText = this.props.filterText;
      const inStockOnly = this.props.inStockOnly;

      const rows = [];
      let lastCategory = null;

      this.props.products.forEach((product) => {
        if (product.name.indexOf(filterText) === -1) {
          return;
        }
        if (inStockOnly && !product.stocked) {
          return;
        }
        if (product.category !== lastCategory) {
          rows.push(
            <ProductCategoryRow
              category={product.category}
              key={product.category} />
          );
        }
        rows.push(
          <ProductRow
            product={product}
            key={product.name}
          />
        );
        lastCategory = product.category;
      });

      return (
        <table>
          <thead>
            <tr>
              <th>Name</th>
              <th>Price</th>
            </tr>
          </thead>
          <tbody>{rows}</tbody>
        </table>
      );
    }
  }
  // 検索領域
  class SearchBar extends React.Component {
    constructor(props) {
      super(props);
      this.handleFilterTextChange = this.handleFilterTextChange.bind(this);
      this.handleInStockChange = this.handleInStockChange.bind(this);
    }
    
    handleFilterTextChange(e) {
      this.props.onFilterTextChange(e.target.value);
    }
    
    handleInStockChange(e) {
      this.props.onInStockChange(e.target.checked);
    }
    
    render() {
      return (
        <form>
          <input
            type="text"
            placeholder="Search..."
            value={this.props.filterText}
            onChange={this.handleFilterTextChange}
          />
          <div>
            <input
              type="checkbox"
              checked={this.props.inStockOnly}
              onChange={this.handleInStockChange}
            />
            {' '}
            Only show products in stock
          </div>
        </form>
      );
    }
  }
  // イベント処理
  class FilterableProductTable extends React.Component {
    constructor(props) {
      super(props);
      this.state = {
        filterText: '',
        inStockOnly: false
      };
      
      this.handleFilterTextChange = this.handleFilterTextChange.bind(this);
      this.handleInStockChange = this.handleInStockChange.bind(this);
    }

    handleFilterTextChange(filterText) {
      this.setState({
        filterText: filterText
      });
    }
    
    handleInStockChange(inStockOnly) {
      this.setState({
        inStockOnly: inStockOnly
      })
    }

    render() {
      return (
        <div>
          <SearchBar
            filterText={this.state.filterText}
            inStockOnly={this.state.inStockOnly}
            onFilterTextChange={this.handleFilterTextChange}
            onInStockChange={this.handleInStockChange}
          />
          <ProductTable
            products={this.props.products}
            filterText={this.state.filterText}
            inStockOnly={this.state.inStockOnly}
          />
        </div>
      );
    }
  }
  // 商品データ
  const PRODUCTS = [
    {category: 'Sporting Goods', price: '$49.99', stocked: true, name: 'Football'},
    {category: 'Sporting Goods', price: '$9.99', stocked: true, name: 'Baseball'},
    {category: 'Sporting Goods', price: '$29.99', stocked: false, name: 'Basketball'},
    {category: 'Electronics', price: '$99.99', stocked: true, name: 'iPod Touch'},
    {category: 'Electronics', price: '$399.99', stocked: false, name: 'iPhone 5'},
    {category: 'Electronics', price: '$199.99', stocked: true, name: 'Nexus 7'}
  ];

  ReactDOM.render(
    <React.StrictMode>
      <FilterableProductTable products={PRODUCTS} />
    </React.StrictMode>,
    document.getElementById('root')
  );
</script>
