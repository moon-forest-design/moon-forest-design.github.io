ボタンの色が反転するホバーアニメ

コード概要

要点

今回は、ボタンの色を変えるホバーアニメーションです。前回のボタンを透過するホバーアニメとおおよそ内容は同じです。ホバー時の処理のみ違います。ポイントは以下ようです。

  • ホバーされたら、文字色と背景色を反転表示します。また、見やすくするために枠線を表示し、文字の太さも変更しています。
ホバーアニメ「マウスをのせると色が変わります」

コード

HTML


<div class="hoverAnime">
  <a class="hoverAnime__link" href="">ホバーアニメ「マウスをのせると色が変わります」</a>
</div>

CSS


.hoverAnime {
  text-align: center;
}
.hoverAnime__link {
  padding: .6em 3em;
  color: #fffffc;
  background-color: #69821b;
  box-shadow: 2px 2px 3px 0 rgba(0, 0, 0, .5);
  border-radius: 4px;
  transition: .5s;
}
.hoverAnime__link:hover {
  font-weight: bold;
  border: 1px solid #69821b;
  color: #69821b;
  background-color: #fffffc;
}

あとがき

  • ボタンを透過するホバーアニメーションと同様に、押下（クリック）できる要素であると気づきやすくさせるメリットがあります。
  • お問い合わせメールの送信ボタンなど、特別なボタンに利用されることが多いようです。
