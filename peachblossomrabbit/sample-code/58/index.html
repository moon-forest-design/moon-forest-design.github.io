Gridレイアウト入門その3

コード概要

要点

今回は Gridレイアウトの grid-column-startと grid-column-endを試します。仮に div要素（A、B、C）を、次のようにレイアウトしたい場合です。ポイントは以下ようです。

  • gridコンテナ化します。2行目を 2カラムにしたいので、「grid-template-columns: repeat(2, 1fr);」と指定します。
  • div要素（A）は、1カラムなので、「grid-column-start: 1;」と「grid-column-end: 3;」を指定します。列の間に 1本ずつ線を引く場合、3本線をひく形になります。ゆえに、開始（start）に 1、終了（end）に 3を指定しているわけです。
  • div要素（B）は、2カラムの左側なので、「grid-column-start: 1;」と「grid-column-end: 2;」を指定します。これも考え方は同様です。開始（start）に 1、終了（end）に 2を指定するわけです。
  • div要素（C）は、2カラムの右側なので、「grid-column-start: 2;」と「grid-column-end: 3;」を指定します。これも同様。

コード

HTML


<div class="gridDemo">
  <div class="gridDemo__a">A</div>
  <div class="gridDemo__b">B</div>
  <div class="gridDemo__c">C</div>
</div>

CSS


.gridDemo {
  display: grid;
  gap: 10px;
  grid-template-columns: repeat(2, 1fr);
}
.gridDemo__a {
  grid-column-start: 1;
  grid-column-end: 3;
}
.gridDemo__b {
  grid-column-start: 1;
  grid-column-end: 2;
}
.gridDemo__c {
  grid-column-start: 2;
  grid-column-end: 3;
}

編集後記

あとがき

  • 「grid-column-start」と「grid-column-end」は、あわせて「grid-column」という形で書くこともできます。例えば、「grid-column-start: 1;」と「grid-column-end: 2;」であれば、あわせて「grid-column: 1 / 2;」と書くことができます。
