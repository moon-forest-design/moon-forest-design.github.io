Gridレイアウト入門その2

コード概要

要点

今回は、Gridレイアウトです。先日、サムネイル画像を並べるだけの簡単なGridレイアウトを試してみましたが、さらにそのコードを改変します。ポイントは以下ようです。

  • 前回の Gridレイアウトと同様に、パソコンサイズは 3列、タブレットサイズは 2列、スマートフォンサイズは 1列にします。ただし、今回はメディアクエリを使いません。
  • 前回は「grid-template-columns: repeat(3, 1fr);」のように書いた部分を、「grid-template-columns: repeat(auto-fit, minmax(315px, 1fr));」とします。何列にするかは自動（auto-fit）でうまいことやってもらい、画像サイズの最小値を 315px、最大値を autoとしています。

コード

HTML


<div class="grid">
  <div class="grid__item"><img class="grid__img" src="family1.jpg"></div>
  <div class="grid__item"><img class="grid__img" src="family2.jpg"></div>
  <div class="grid__item"><img class="grid__img" src="family3.jpg"></div>
  <div class="grid__item"><img class="grid__img" src="family4.jpg"></div>
  <div class="grid__item"><img class="grid__img" src="family5.jpg"></div>
</div>

CSS


.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(315px, auto));
  gap: 20px;
}
