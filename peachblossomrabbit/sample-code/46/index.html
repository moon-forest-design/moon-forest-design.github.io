リンクの下線が伸びるホバーアニメ

コード概要

要点

今回も、リンク下線を表示するホバーアニメーションです。前回は、リンク下線をふわっと表示するホバーアニメでしたが、今回は文字列の上にマウスポインターをのせる（ホバーする）と、下線が伸びるタイプです。ポイントは以下ようです。

  • a要素に疑似要素（::after）を作成し、この疑似要素にホバーアニメをほどこします。
  • ホバーされない状態では、非表示（opacity: 0;）にしてあります。幅も 0です。（width: 0%;）
  • 「position: absolute;」を使って、左下すみに置きます。（left: 0; 及び bottom: 0;）
  • ホバーされたら、表示（opacity: 1;）し、幅を大きくします。（width: 100%;）
  • アニメーションの時間は、0.7sです。（transition: .7s;）
ホバーアニメ「マウスをのせると下線を表示します」

コード

HTML


<div class="hoverAnime">
  <a class="hoverAnime__link" href="">ホバーアニメ「マウスをのせると下線を表示します」</a>
</div>

CSS


.hoverAnime {
  text-align: center;
}
.hoverAnime__link {
  display: inline-block;
  position: relative;
}
.hoverAnime__link::after {
  width: 0%;
  height: 2px;
  position: absolute;
  left: 0;
  bottom: 0;
  content: "";
  border-bottom: 2px solid #302833;
  opacity: 0;
  transition: .7s;
}
.hoverAnime__link:hover::after {
  width: 100%;
  opacity: 1;
}
