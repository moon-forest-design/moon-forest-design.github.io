React入門4

コード概要

要点

  • 「JSX」を使った書き方その1。()かっこ。例えば、return ();にすれば、複数行の要素を書いてもいい。ただし、returnは単一要素にしないとエラーになるので、div要素で囲んである。
  • 「JSX」を使った書き方その2。{}かっこ。いろいろな用途に使えるようだ。イメージとしては、JavaScriptの ${}や ``に近いかも。例えば、HTML内で変数を使いたいときに、constや letで宣言した値を、{変数名}と書くことで参照できるようになる。また、関数を呼び出すこともできる。
  • 「JSX」を使った書き方その3。class属性はclassNameと書く。
  • 「JSX」を使った書き方その4。CSSをあてるときは、style={{}}として、JSON形式で書く。

コード

HTML


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  function App() {
    const prefix = "Reactテスト：";
    return (
      <div className="reactText">
        <p style={{ fontWeight: 'bold' }}>{prefix}JSXを使うと</p>
        <p>{prefix}複数行書ける</p>
      </div>
    );
  }
  ReactDOM.render(
    <React.StrictMode>
      <App />
    </React.StrictMode>,
    document.getElementById('root')
  );
</script>
