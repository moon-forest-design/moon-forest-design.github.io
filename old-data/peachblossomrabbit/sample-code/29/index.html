アコーディオン Q&A

コード概要

要点

今回は、アコーディオン Q&Aについて書きます。ポイントは以下ようです。

  • Q&Aは、dl要素で組みます。dt要素に .question（質問）、dd要素に .answer（回答）という class名をつけてあります。
  • クリックできる場所だと分かりやすいように、.questionには「cursor: pointer;」を指定してあります。
  • .answerは、初期状態では非表示にしたいので「display: none;」を指定してあります。
  • jQueryを使用しています。クリックされたら、一度 hide()メソッドですべて閉じます。そして、slideToggle()メソッドでクリックされた要素（this）を開く仕組みです。
  • next()メソッドは、クリックされた要素（this）の弟要素に対して slideToggle()をかけるために使用しています。

「アコーディオン Q&A」に関するご質問と回答

Q1　アコーディオン Q&Aとは？
A1　楽器のアコーディオンのように伸縮するタイプの Q&Aです。よくある質問などに利用できます。
Q2　なぜアコーディオン型にするのですか？
A2　スクロールせず、質問項目だけを一瞥して確認できる利点があります。求める情報に辿り着きやすくなります。
Q3　アコーディオン Q&Aの導入に必要なものは？
A3　JavaScript等によるプログラミングが必要です。jQueryを使うと簡便に制作できます。
Q4　注意点はありますか？
A4　クリックの必要性に気づいてもらう工夫をしておくと良いでしょう。
Q5　コードを見せてください。
A5　下記、ご参照くださいますようお願い申しあげます。

CSS


.question {
  padding: 1em;
  cursor: pointer;

jQuery


$(function() {
  $(".question").on("click", function() {
    $(".answer").hide("slow");
    $(this).next().slideToggle("slow");
  });
});
