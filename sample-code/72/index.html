React入門
「propsなしの引数」

コード概要

要点

  • 引数を受け取るときに、関数を ()ではなく ({ ... })とすると、「props.」をつけなくても良くなるらしい。これは Reactの話ではなく、JavaScriptの分割代入。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  const Hello = ({ name, color }) => (
    <div>
      <p style={{ color }}>Hello {name}</p>
    </div>
  );
  const TextView = () => (
    <div>
      <Hello name="Rabbit" color="red" />
      <Hello name="Cat" color="blue" />
      <Hello name="Raccoon dog" color="green" />
    </div>
  );
  ReactDOM.render(
    <React.StrictMode>
      <TextView />
    </React.StrictMode>,
    document.getElementById('root')
  );
</script>
