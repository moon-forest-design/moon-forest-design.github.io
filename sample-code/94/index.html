React入門
「関数コンポーネントの機能的分割」

コード概要

要点

  • React公式の「コンポーネントと props」で紹介されているコード。関数コンポーネントの中で機能的に分割できるものは、関数として抽出するほうがいいよというお話。関数 Commentの中にすべて書いてしまうのではなく、UserInfoの部分を外に出し、さらにその中のAvatarすらも外に切り出すくらいで良いと。細かくコンポーネント化する。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  function formatDate(date) {
    return date.toLocaleDateString();
  }

  function Avatar({name, avatarUrl}) {
    return (
      <img
        className="Avatar"
        src={avatarUrl}
        alt={name}
      />
    );
  }

  function UserInfo({name, avatarUrl}) {
    return (
      <div className="UserInfo">
        <Avatar name={name} avatarUrl={avatarUrl} />
        <div className="UserInfo-name">{name}</div>
      </div>
    );
  }

  function Comment({date, text, author}) {
    return (
      <div className="Comment">
        <UserInfo name={author.name} avatarUrl={author.avatarUrl} />
        <div className="Comment-text">{text}</div>
        <div className="Comment-date">
          {formatDate(date)}
        </div>
      </div>
    );
  }

  const comment = {
    date: new Date(),
    text: 'I hope you enjoy learning React!',
    author: {
      name: 'peachblossomrabbit',
      avatarUrl: 'logo-peachblossomrabbit.png',
    },
  };

  const reactRender = () => ReactDOM.render(
    <React.StrictMode>
      <Comment
        date={comment.date}
        text={comment.text}
        author={comment.author}
      />
    </React.StrictMode>,
    document.getElementById('root')
  );
  reactRender();
</script>
