画像3枚を並べてフル画面表示する方法

コード概要

要点

今回は、画像3枚を均等に並べてフル画面表示する方法について書きます。ポイントは以下ようです。

  • ul要素にリストを用意し、画像を 3枚入れます。ul要素には class名 .mainVisualを指定し、flexをかけます。
  • 各 li要素（.mainVisual__item）の幅は、calc()で算出します。これで横幅いっぱいに表示します。
  • 各 img要素（.mainVisual__img）の高さは 100vhを指定し、「object-fit: cover;」をあてることで縦の高さいっぱいで表示できます。
  • スマートフォンでは、3枚の画像を横並びにせず、縦に並ぶ形にしました。レスポンシブ対応。

コード

HTML


<ul class="mainVisual">
  <li class="mainVisual__item"><img class="mainVisual__img" src="fruits-bananas.jpg"></li>
  <li class="mainVisual__item"><img class="mainVisual__img" src="fruits-strawberries.jpg"></li>
  <li class="mainVisual__item"><img class="mainVisual__img" src="fruits-pear.jpg"></li>
</ul>

CSS


.mainVisual {
  display: flex;
  list-style-type: none;
}
.mainVisual__item {
  width: calc(100% / 3);
}
.mainVisual__img {
  height: 100vh;
  object-fit: cover;
}
@media screen and (max-width: 576px) { /* スマホ用 */
  .mainVisual {
    flex-direction: column;
  }
  .mainVisual__item {
    width: 100%;
  }
  .mainVisual__img {
    height: calc(100vh / 3);
  }
}

編集後記

あとがき

  • calc()は、割り切れない値でもうまいこと処理してくれるらしいのでありがたいです。太古の昔みたいに「33%でいいか、しょうがないよ」みたいなことをしなくて良いので助かります。
