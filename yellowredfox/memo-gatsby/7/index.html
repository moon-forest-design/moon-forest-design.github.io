ブログコンテンツの表示方法

概要

要点

  1. 「ファイル名の表示方法」の続き。今回はファイル名ではなく、コンテンツ内容を表示する。まず、/blog下に作成した mdxファイルの中に、Markdow記法でフロントマターやコンテンツを書いておく。
  2. MDXトランスフォーマープラグイン gatsby-plugin-mdx及び、依存関係にあるプラグイン（@mdx-js/mdx、@mdx-js/react）をインストールする。
  3. gatsby-config.jsファイルに、gatsby-plugin-mdxの設定を追加する。
  4. GraphiQLインターフェースでクエリを試す。allMdxの nodesの frontmatterの dateと title、そして keyに使用するための id、コンテンツ内容 body、さらに降順に sortするための設定を入力し、実行ボタンを押す。上述にて作成した mdxコンテンツがうまいこと表示されれば OK。
  5. src/pages/下に作成した blog.jsを次のようにする。
    • MDXRendererを importする。
    • Array.mapを使って、mdxコンテンツの内容をファイルの数だけ繰り返し記述する。
    • 上述 GraphiQLインターフェースで試したクエリを定数 queryへ代入する。

手順

mdxファイル内容（例）


// my-first-post.mdx
---
title: "My First Post"
date: "2021-07-23"
---

This is my first blog post! Isn't it *great*?

Some of my **favorite** things are:

* Petting dogs
* Singing
* Eating potato-based foods


// another-post.mdx
---
title: "Another Post"
date: "2021-07-24"
---

Here's another post! It's even better than the first one!


// yet-another-post.mdx
---
title: "Yet Another Post"
date: "2021-07-25"
---

Wow look at all this content. How do they do it?

コマンドライン


// gatsby-plugin-mdx、@mdx-js/mdx、@mdx-js/reactをインストール
npm install gatsby-plugin-mdx @mdx-js/mdx @mdx-js/react

gatsby-config.js


// gatsby-config.jsファイルに設定を追加
module.exports = {
  siteMetadata: {
    title: "サイト名",
  },
  plugins: [
    "gatsby-plugin-image",
    "gatsby-plugin-sharp",
    {
      resolve: "gatsby-source-filesystem",
      options: {
        name: `blog`,
        path: `${__dirname}/blog`,
      }
    },
    "gatsby-plugin-mdx",
  ],
};

GraphiQLインターフェース


query MyQuery {
  allMdx(sort: {fields: frontmatter___date, order: DESC}) {
    nodes {
      frontmatter {
        date(formatString: "MMMM D, YYYY")
        title
      }
      id
      body
    }
  }
}

blog.js


import * as React from 'react'
import { graphql } from 'gatsby'
import { MDXRenderer } from 'gatsby-plugin-mdx'
import Layout from '../components/layout'

const BlogPage = () => {
  return (
    <Layout pageTitle="My Blog Posts">
      {
        data.allMdx.nodes.map((node) => (
          <article key={node.id}>
            <h2>{node.frontmatter.title}</h2>
            <p>Posted: {node.frontmatter.date}</p>
          </article>
        ))
      }
    </Layout>
  )
}

export const query = graphql`
query {
  allMdx(sort: {fields: frontmatter___date, order: DESC}) {
    nodes {
      frontmatter {
        date(formatString: "MMMM D, YYYY")
        title
      }
      id
      body
    }
  }
}
`

export default BlogPage
