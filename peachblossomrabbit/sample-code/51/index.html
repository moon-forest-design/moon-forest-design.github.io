footerを最下部に表示する方法

コード概要

要点

今回は、footerを最下部に表示する方法です。コンテンツが多ければ、自然と下の方へ配置されて問題ないのですが、コンテンツが少ないと中途半端な位置にきてしまい、見栄えが悪くなることがあります。そのような状態でもうまいこと表示するための方法です。

コンテンツが多ければとくに何もなく、コンテンツが少ない場合にのみ、footer要素をブラウザ画面の底に表示します。ポイントは以下ようです。

  • body要素に「min-height: 100vh;」とすることで、最低でも画面いっぱいの高さで表示します。
  • flexを使います。body要素に対して「flex-direction: column;」することでフレックスコンテナの主軸を列（column）方向にします。
  • main要素に「flex: 1;」をかけます。flexプロパティ（flex-grow）を利用することで main要素の高さを必要に応じて伸ばします。

コード

HTML


<html>
<body>
  <header>～中略～</header>
  <main>～中略～</main>
  <footer>～中略～</footer>
</body>
</html>

CSS


body {
  min-height: 100vh;
  display: flex;
  flex-direction: column;
}
main {
  flex: 1;
}
