clip-pathを使った三角リストマーカーの作り方

コード概要

要点

今回は、clip-pathを使った三角リストマーカーの作り方です。前回のborderを使った方法は、ほんの少しだけ理解するために時間を要する内容でしたが、clip-pathを使った方法はとても理解しやすいものです。ポイントは以下ようです。

  • 三角形を作るための領域が必要です。三角形の大きさは width・heightで指定します。
  • 三角形を描くために、clip-pathの polygonを利用します。「clip-path: polygon(0 0, 0% 100%, 100% 50%);」とすれば、三角形を描くことができます。polygonで指定しているのは、最初のXY座標（0 0）、二番目のXY座標（0% 100%）、三番目のXY座標（100% 50%）です。
  • あとは、background-colorで色を塗り、位置を「position: absolute;」で調整してあげれば、三角リストマーカーの完成です。

コード

HTML


<ul class="triangleList">
  <li>項目1</li>

</ul>

CSS


.triangleList {
  list-style-type: none;
}
.triangleList li {
  position: relative;
  padding-left: 18px;
}
.triangleList li:before {
  display: block;
  content: "";
  position: absolute;
  top: 5px;
  left: 0;
  border-top: 7px solid transparent;
  border-right: 7px solid transparent;
  border-bottom: 7px solid transparent;
  border-left: 7px solid #1e50a2;
}
