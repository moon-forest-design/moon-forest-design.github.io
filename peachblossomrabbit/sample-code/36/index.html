ベーシックな一覧表の作り方

コード概要

要点

今回は、ベーシックな一覧表（table）の作り方です。非常にオーソドックスなただの一覧表です。ポイントは以下ようです。

  • 「border-collapse: collapse;」を指定し、セルの間隔をつぶす。
  • table要素の上と左に線を引き、th・td要素の右と下に線を引けば、重なることなく済む。

コード

HTML


<table class="table">
  <thead>
    <tr>
      <th>見出し</th><th>見出し</th><th>見出し</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>データ</td><td>データ</td><td>データ</td>
    </tr>
    <tr>
      <td>データ</td><td>データ</td><td>データ</td>
    </tr>
    <tr>
      <td>データ</td><td>データ</td><td>データ</td>
    </tr>
  </tbody>
</table>

CSS


.table {
  width: 100%;
  border-collapse: collapse;
  text-align: center;
  border-top: 1px solid #302833;
  border-left: 1px solid #302833;
}
.table th,
.table td {
  padding: .5em 1em;
  border-right: 1px solid #302833;
  border-bottom: 1px solid #302833;
}
.table th {
  background-color: #ffccaa;
}
