React入門
「class構文によるToggleボタン」

コード概要

要点

  • class構文で定義したイベント処理のサンプル Toggleボタン。
  • onClick={this.handleClick} のように ()を末尾に付けずにメソッドを参照する場合は、そのメソッドは bind()しておく必要がある。コンストラクタ内で書かれている this.handleClickの thisは関数外なので bind()しなければならない。
    • bind()が手間であれば、アロー関数を使う方法もある。例：onClick={() => this.handleClick()}
  • setState()で、コンポーネントの更新を要求する。ただし、処理効率をより良くするため、即時に更新するとは限らない。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  class Toggle extends React.Component {
    constructor(props) {
      super(props);
      this.state = {isToggleOn: true};
      // bind()は、コールバックで thisを機能させるために必要
      this.handleClick = this.handleClick.bind(this);
    }

    handleClick() {
      // setState()で、コンポーネントの更新を要求する
      this.setState(prevState => ({
        isToggleOn: !prevState.isToggleOn
      }));
    }

    render() {
      return (
        <button onClick={this.handleClick}>
          {this.state.isToggleOn ? 'ON' : 'OFF'}
        </button>
      );
    }
  }

  ReactDOM.render(
    <Toggle />,
    document.getElementById('root')
  );

</script>
