Grobal CSS

概要

要点

  1. src/styles/下に grobal.cssファイルを作成する。とりあえず、.containerの指定だけ記述。
  2. プロジェクトディレクトリ直下に、gatsby-browser.jsを作成する。上述の grobal.cssを importする。
  3. src/components/下の layout.jsに以下指定。importは不要。
    • div要素に className属性で指定。例：<div className={grobal.container}>

手順

grobal.css


.container {
  margin: auto;
  max-width: 500px;
  font-family: sans-serif;
}

gatsby-browser.js


import "./src/styles/grobal.css";

layout.js


import * as React from 'react';
import { Link } from 'gatsby';

const Layout = ({ pageTitle, children }) => {
  return (
    <div className={grobal.container}>
      <title>{pageTitle}</title>
      <nav>
        <ul>
          <li>
            <Link to="/">Home</Link>
          </li>
          <li>
            <Link to="/about">About</Link>
          </li>
        </ul>
      </nav>
      <main>
        <h1>{pageTitle}</h1>
        {children}
      </main>
    </div>
  )
};

export default Layout;

編集後記

あとがき

  • grobal.cssに要素型セレクタだけを指定するのであればとくに気にせず書けばいいけれど、classセレクタで書く場合はあまり好ましくない手間が入る。例えば、grobal.cssで「.container」と書いたら、layout.jsで「className={container}」と書きたくなるが、これはエラーになる。「className={grobal.container}」としてあげる必要があるようだ。見た目も悪い。いかがなものか。何か他に良い方法があるのかもしれない。
