月の森デザイン

月の森デザインは、Webデザイナー「イールド（yield）」が運営するサイトです。ホームページを制作しています。

ツキノラボ

このサイトです。TEST

ツキノメモ

技術メモです。GatsbyJSで制作しました。

習作サイト

FizzBuzzy

FizzBuzzを Webゲームにしました。「Fizz」や「Buzz」などのボタンを押して、落ちてくる数字を消すだけの簡単なゲームです。Canvasを使用しています。

カラーコードつくーる

Sass用のコードリストを作成するためのツールです。変数名と 16進数カラーコードを出力します。色は日本の伝統色（和色）を使用。

日付リストつくーる

日付リストを作成するWebツールです。1ヶ月分の日付を、テキストエディタなどに貼り付けて使用します。作業ログを書くときに便利なので制作しました。

simpleCalculatorβ

電卓のWebツールです。四則演算しかできない少機能シンプル設計。残念ながら、小数点計算が怪しい β（ベータ）版。

JavaScript superStars

GitHubの人気リポジトリ 年間 Top10を表示する Webサイトです。カテゴリは JavaScript。スター数が多いリポジトリ順で表示。Fetch APIを使用しました。

qNewest

Qiitaの最新記事を表示する Webサイトです。HTML・CSSなどのカテゴリごとに表示できます。XMLHttpRequestを使用しました。

ストップウォッチ

ストップウォッチの Webツールです。ラップタイム機能付き。一定区間ごとの経過時間を測ることができます。

ToDoList やること帳

ToDoリストの Webツールです。追加、編集・更新、削除、それと作業が済んだ項目にはチェックを入れて終了が分かりやすくなる機能もあります。

ポモドーロ・鳩タイマー

ポモドーロ・テクニックをもとにして制作した Webツールです。時間がくると、鳩が教えてくれる仕様です。

フィクション ドーナツ

実在しないドーナツ専門店の Webサイトを制作。コンテンツのライティングに注力しました。

フィクション歯科医院

実在しない歯科医院の Webサイトを制作。トップページのファーストビューには、jQueryと CSSアニメーションを使用しました。
