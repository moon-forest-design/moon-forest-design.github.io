ページのスクロールを促す方法

Please Scroll

コード概要

要点

今回は、ページのスクロールを促す方法を書き残しておきます。スクロールされるまでメッセージ「Please Scroll」がバウンドし続けるアニメーション付きです。仕様は以下ようです。

  • 事前に、メッセージ「Please Scroll」を、CSSで非表示にしておきます。
  • 3秒経っても、ユーザーがページをスクロールしなければ、メッセージ「Please Scroll」を jQueryを使って表示します。
  • メッセージ「Please Scroll」は、CSSアニメーションによりバウンドします。
  • ユーザーがページをスクロールしたら、メッセージ「Please Scroll」を jQueryを使って非表示にします。

何をしているのだか分かりにくい点があるとすれば、CSSの「transform: translate(-50%, -50%);」部分でしょうか。これをすることで、テキスト幅の半分だけずらして中央揃えにしています。

コード

HTML


<p id="pleaseScroll" class="pleaseScroll animationBounce" data-subarrow="▼">Please Scroll</p>

CSS


.pleaseScroll {
 display: none;
 font-size: 2rem;
 width: 200px;
 margin: 0 50%;
 position: absolute;
 bottom: 30px;
 transform: translate(-50%, -50%);
 text-align: center;
 font-weight: bold;
 color: #192f60;
}
.pleaseScroll::after {
 display: block;
 content: attr(data-subarrow);
}
.animationBounce {
　animation: bounce 2s ease infinite normal 0s none running;
}
@keyframes bounce {
　0%, 100%, 20%, 50%, 80% {
　　transform: translate(-50%, 0px);
　}
　40% {
　　transform: translate(-50%, -30px);
　}
　60% {
　　transform: translate(-50%, -15px);
　}
}

jQuery


$(function() {
 // スクロールされないとき 「Please Scroll」を表示
 setTimeout(function() { // 3秒待つ
  if ($(window).scrollTop() < 10) { // 3秒経っても上端から10px以上スクロールしない場合
  $("#pleaseScroll").fadeIn(); // 「Please Scroll」を表示しスクロールを促す
  }
 }, 3000);

 // スクロールされたら「Please Scroll」を非表示
 $(window).scroll(function() {
  $("#pleaseScroll").fadeOut();
 });
});
