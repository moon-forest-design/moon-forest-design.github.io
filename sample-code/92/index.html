React入門
「class構文によるいいねボタン」

コード概要

要点

  • React公式の「既存のウェブサイトに React を追加する」で紹介されている いいねボタンを参考にして、一部 JSXで書き直した。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  class LikeButton extends React.Component {
    constructor(props) {
      super(props);
      this.state = { liked: false };
    }

    render() {
      if (this.state.liked) {
        return 'You liked this !';
      }

      return (
        <button onClick={() => this.setState({ liked: true })}>
          Like
        </button>
      );
    }
  }

  const reactRender = () => ReactDOM.render(
    <React.StrictMode>
      <LikeButton />
    </React.StrictMode>,
    document.getElementById('root')
  );
  reactRender();
</script>
