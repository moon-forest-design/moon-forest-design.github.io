GatsbyJSで Header・Footerコンポーネントを作成する方法

投稿日：2021 / 10 / 14

概要

前回作成した Layoutコンポーネントの続きです。今回は、GatsbyJSで Headerコンポーネントを作成します。

Layoutコンポーネントに書かれているコードの中には、Headerコンポーネントとして分離するほうが好ましい部分があります。Header.jsというファイルを作り、書き出してしまいましょう。そして、Layout.jsから Header.jsを呼び出す形にします。

ついでなので Footerコンポーネントも作成します。

開発環境

  • Windows10
  • Node.js: 14.18.0
  • npm: 6.14.15
  • gatsby-cli: 3.14.1
  • GatsbyJS: 4.4.0
    • Starter「gatsby-starter-hello-world | Gatsby」を利用。
    • チュートリアル（v4）を参考にして制作。

作業手順

1：header.jsを作成する

  • src/components/下に header.jsファイルを作成します。ヘッダーに書いたほうが良い部分を Layoutコンポーネントから抽出し、独り立ちさせます。
  • header要素を記述し、その中に title要素と nav要素を移動しましょう。
  • title要素のページタイトルを表示するために、layout.jsから pageTitleを受け取る必要があります。「const Header = ({ pageTitle }) => { ... };」と書いて、引数を受け取ります。
src/components/header.js
import * as React from 'react';
import { Link } from 'gatsby';

const Header = ({ pageTitle }) => {
  return (
    <header>
      <title>{pageTitle}</title>
      <nav>
        <ul>
          <li>
            <Link to="/">Home</Link>
          </li>
          <li>
            <Link to="/about">About</Link>
          </li>
        </ul>
      </nav>
    </header>
  )
};

export default Header;

2：layout.jsを編集する

  • src/components/下にある layout.jsファイルを編集します。まずは、header.jsへ移動したコードを削除しましょう。
  • header.jsを読み込む必要があるので、「import Header from './header';」と書いてインポートします。<Header />と書くことで、header.jsを読み込むことができます。また、引数として受け取った pageTitleを header.jsへ渡すことも忘れてはいけません。
  • Linkを使わなくなったので、Linkのインポートは削除します。
src/components/layout.js
import * as React from 'react';
import Header from './header';

const Layout = ({ pageTitle, children }) => {
  return (
    <div>
      <Header pageTitle={pageTitle} />
      <main>
        <h1>{pageTitle}</h1>
        {children}
      </main>
    </div>
  )
};

export default Layout;

3：footer.jsを作成する

  • src/components/下に footer.jsファイルを作成します。
src/components/footer.js
import * as React from 'react';

const Footer = () => {
  return (
    <footer>
      <p><small>Copyright &copy; 2021 （著作権者） All Rights Reserved.</small></p>
    </footer>
  )
};

export default Footer;

4：layout.jsを編集する

  • src/components/下にある layout.jsファイルを編集します。footer.jsを読み込むために、「import Footer from './footer';」と書いてインポートします。<Footer />と書くことで、footer.jsを読み込むことができます。
src/components/layout.js
import * as React from 'react';
import Header from './header';
import Footer from './footer';

const Layout = ({ pageTitle, children }) => {
  return (
    <div>
      <Header pageTitle={pageTitle} />
      <main>
        <h1>{pageTitle}</h1>
        {children}
      </main>
      <Footer />
    </div>
  )
};

export default Layout;

5：表示を確認する

  • 「gatsby develop」し、ブラウザで問題なく表示されたら OKです。
