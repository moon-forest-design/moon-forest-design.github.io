コンポーネントで引数を使う

  • 「name=""」とすれば、プロパティ値を関数に渡すことができる。引数名は propsとし、関数内で「{props.name}」と書けば参照できる。
  • propsは、「プロパティ」を意味する言葉。

コード

HTML（コンポーネントを使う）


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  function App() {
    return <p>Reactテスト：コンポーネントを使う</p>;
  }
  ReactDOM.render(
    <React.StrictMode>
      <App />
    </React.StrictMode>,
    document.getElementById('root')
  );
</script>

HTML（コンポーネントで引数を使う）


<div id="root2" class="reactRoot"></div>
<script type="text/babel">
  function App(props) {
    return <p>Reactテスト：コンポーネントで{props.name}を使う</p>;
  }
  ReactDOM.render(
    <React.StrictMode>
      <App name="引数" />
    </React.StrictMode>,
    document.getElementById('root2')
  );
</script>
