object-fitで画像をフル表示する方法

コード概要

要点

今回は、CSSの object-fitプロパティを使って、画像を画面いっぱいに表示する方法について書きます。前回は、backgroundを使用して、いわゆるヒーローイメージを設定しましたが、今回は object-fitプロパティを使います。ポイントは以下ようです。

  • img要素を使って画像を表示する。img要素を指定することで、画像検索の対象となりやすくなる。
  • widthに 100%、heightに 100vhを指定。画像の元サイズが幅 1920pxなので、max-widthで 1920pxを指定。max-widthを指定することで、画質をたもつ。
  • object-fitに coverを指定。coverを指定することで、レスポンシブ対応となる。スマホやタブレットで表示しても歪むことがない。

コード

HTML


<div class="mainVisual">
  <img class="mainVisual__img" src="mainVisual.jpg">
</div>

CSS


.mainVisual__img {
  width: 100%;
  max-width: 1920px;
  height: 100vh;
  object-fit: cover;
}

編集後記

あとがき

  • object-positionプロパティを使えば、画像の位置を柔軟に変更することも可能です。
  • object-fitプロパティには他にも、video要素（動画）にも活用できるという利点があります。
  • IE対応のサイトにしたいときは、backgroundで表示する方法を利用するほうが良いかもしれません。
