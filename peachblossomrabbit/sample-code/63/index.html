ヘッダーを画面いっぱいに表示する方法
「height: 100vh;」を使用

コード概要

要点

ヘッダーをウィンドウ画面いっぱいに表示して、タイトルを上下左右中央揃えにする方法を書き残しておきます。今回は「height: 100vh;」を使います。CSSのポイントは以下ようです。

  • header要素に、「height: 100vh;」を指定する。
  • 必要となる箇所まで高さを継承する。例えば今回の場合は、サイトタイトルの親要素（.header__container）まで header要素の heightの値（100%）を inheritして継承した。
  • サイトタイトルの上下左右の中央揃えは好きな方法でかまわない。今回は、親要素（.header__container）に flexを指定した。

コード

HTML


<header class="header">
　<div class="header__container">
　　<h1 class="siteTitle">ヘッダーを画面いっぱいに表示する方法「height: 100%;」を使用</h1>
　</div>
</header>

CSS


.header {
  height: 100vh;
}
.header__container {
　height: inherit;
　display: flex;
　justify-content: center;  
　align-items: center;
}

編集後記

あとがき

100vhの方法で行う場合は、スマホ上部に表示されるアドレスバーを考慮する必要があるようです。
