ECMAScript modulesの
import / exportを試す

コード概要

要点

  • ECMAScript modulesの import / exportを試すには、サーバを起てる必要がある。サーバを起てずにローカル環境で試すと、クロスオリジンリソース共有（CORS）エラーが出てしまうためだ。今回は、VSCodeの拡張 Live Serverを使用した。
    • Live Serverをインストールし、右下に表示される Go Liveボタンをクリックするだけでサーバ環境としてテストできる。
    • 自動でブラウザに表示されると困るときは、設定の「liveServer.settings.NoBrowser」にチェックを入れるといい。例えば、エクスプローラのワークスペースで別のフォルダを開いている場合にはそのフォルダが表示されてしまうようなので、状況によってはおかしな状態になる。関連のないものがブラウザに表示されると戸惑うことになるので、ワークスペースからフォルダを削除するほうがいいかもしれない。
    • ブラウザに表示したい index.htmlファイルを開き、右クリックで「Open with Live Server」を選択するといい。状況によっては手動のほうが混乱がない。「liveServer.settings.NoBrowser」の設定がデフォルトのままで、且つエクスプローラで別のフォルダを開いていると、「Open with Live Server」を選択しても何も起こらないかもしれない。
  • ポイントは以下よう。
    • JavaScriptファイルは 3つ用意した。ファイル名はそれぞれ、main.js、hello.js、goodLuck.jsとした。main.jsは import用。hello.jsは named export用。goodLuck.jsは default export用である。
    • index.htmlで main.jsを読み込むときは、type="module"を指定する必要があるようだ。
    • hello.js（named export用）には、受け取った引数に「Hello」をつけて表示する関数を書いた。named exportで関数を宣言するときは、exportだけをつける。
    • goodLuck.js（default export用）には、受け取った引数に「Good Luck」をつけて表示する関数を書いた。default exportで関数を宣言するときは、export defaultをつける。
    • main.jsでは、hello.js（named export用）とgoodLuck.js（default export用）から、それぞれ関数を importして、関数に引数を渡して使用している。

コード

HTML


<script type="module" src="main.js"></script>

<!-- demo -->
<div id="root" class="demoRoot"></div>

JavaScript：main.js


// named export
import { sayHello } from "./hello.js";
sayHello("Tom");

// named export
// ※：importする関数名は asで変更できる
import { sayHello as h } from "./hello.js";
h("John");

// default export
// ※：{}不要。
// ※：default exportの場合は、sayGoodLuckのように好きなように関数名をつけることができる。
import sayGoodLuck from "./goodLuck.js";
sayGoodLuck("Susan");

JavaScript：hello.js


// named export
export function sayHello(str) {
  // HTMLファイルへ表示
  const root = document.getElementById("root");
  const p = document.createElement('p');
  const text = document.createTextNode(`Hello ${str} !`);
  root.appendChild(p).appendChild(text);

  // コンソール表示
  console.log(`Hello ${str} !`);

}

JavaScript：goodLuck.js


// default export
export default function(str) {
  // HTMLファイルへ表示
  const root = document.getElementById("root");
  const p = document.createElement('p');
  const text = document.createTextNode(`Good Luck ${str} !`);
  root.appendChild(p).appendChild(text);

  // コンソール表示
  console.log(`Good Luck ${str} !`);

}
