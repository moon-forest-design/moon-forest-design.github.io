React入門「CSS」

コード概要

要点

  • CSSを設定する方法1つ目。要素に style属性で指定する。例えば「style={{ color : 'red' }}」のように書く。
  • CSSを設定する方法2つ目。一度変数でまとめて、要素に style属性で指定する。例えば「const colorBlue = { color : 'red' };」とし、「style={colorBlue}」のように書く。
  • CSSを設定する方法3つ目。className属性を使い、外部ファイルで指定。
  • style属性を使う場合は、キャメルケース（camel case）で書く必要がある。例えば background-colorなら backgroundColorと書く。

コード

HTML


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  const colorBlue = { color : 'blue' }; 
  const TextView = () => (
    <div>
      <p>CSSを設定する方法 3つ。</p>
      <ul>
        <li style={{ color : 'red' }}>要素に style属性で直書き</li>
        <li style={colorBlue}>一度変数でまとめて、要素に style属性</li>
        <li className="colorGreen">外部ファイル</li>
      </ul>
    </div>
  );
  ReactDOM.render(
    <React.StrictMode>
      <TextView />
    </React.StrictMode>,
    document.getElementById('root')
  );
</script>
