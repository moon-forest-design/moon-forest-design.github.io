コード概要

要点

  • form要素内のbutton要素をクリックすると、form要素の onSubmit属性に指定してある関数 handleSubmitを呼び出す。関数内では2つの処理を行っている。
  • 1つは、button要素をクリックしたことで発生するデフォルト処理を抑止するために、「e.preventDefault();」をしている。Reactでは falseを返してもデフォルトの動作を抑止することができないため、明示的に preventDefaultを呼び出す必要がある。
  • 2つ目は、クリックされたことが分かるよう「You clicked submit.」とコンソール表示している。

コード

HTML


<div id="root" class="demoRoot"></div>

jsx


  function Form() {
    function handleSubmit(e) {
      // button要素をクリックしたことで発生するデフォルト処理を抑止
      e.preventDefault();
      console.log('You clicked submit.');
    }

    return (
      <form onSubmit={handleSubmit}>
        <button type="submit">Submit</button>
      </form>
    );
  }

  ReactDOM.render(
    <React.StrictMode>
      <Form />
    </React.StrictMode>,
    document.getElementById('root')
  );
