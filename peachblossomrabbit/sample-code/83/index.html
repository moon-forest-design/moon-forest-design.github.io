「コンポーネント間での情報の受け渡し」

コード概要

要点

  • 親コンポーネント（ParentComponent）から、子コンポーネント（ChildComponent）へ引数を受け渡す方法。例えば、親コンポーネントを呼び出すときに「name='山田'」とし、親コンポーネントから子コンポーネントを呼び出すときに「name={this.props.name}」とすれば受け渡すことができる。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  class ChildComponent extends React.Component {
    render() {
      return (
        <div>
          <p>子：{this.props.name} 一郎</p>
        </div>
      )
    }
  };
  class ParentComponent extends React.Component {
    render() {
      return (
        <div>
          <p>親：{this.props.name} 太郎</p>
          <ChildComponent name={this.props.name} />
        </div>
      )
    }
  };
  //コンポーネントの描画
  ReactDOM.render(
    <ParentComponent name='山田' />,
    document.getElementById('root')
  );
</script>
