React入門「イベント処理1」

コード概要

要点

  • Reactのイベント処理は、on系の属性を使用する。（例：onClick属性、onChange属性など）
  • onClick={...}のように書き、中には関数を指定する。関数には、イベントハンドラ（ユーザの操作により発生した処理に対する命令群）を書く。今回のサンプルでは、ボタンがクリックされたら「クリックされた」とコンソール表示するようアロー関数で書いている。

コード

HTML


<script type="text/babel">
  const ClickBtn = () => {
    return (
      <div>
        <button onClick={() => console.log('clicked')}>Click me!</button>
      </div>
    )
  }
  ReactDOM.render(
    <ClickBtn />,
    document.getElementById('root')
  );
</script>

編集後記

あとがき

  • React公式の MAIN CONCEPTを読んでいると、イベント処理あたりから徐々に怪しくなってくる。JavaScriptでよく使用していた addEventListener()は使わない。アロー関数による省略された記述。コールバック関数なども当然使用される。シンプルな形で慣れていけると良い。
