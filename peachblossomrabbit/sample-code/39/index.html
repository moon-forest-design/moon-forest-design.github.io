Swiperの使い方

コード概要

要点

今回は、カルーセルスライダー「Swiper」の使い方です。やることは、前回の「Slick」とそれほど大きく変わるものではありません。ポイントは以下ようです。

  • Swiper本体はダウンロードしてもいいし、CDNを利用してもどちらでも構わない。今回は、CDNを利用した。
  • swiper-bundle.min.cssの後に style.cssを読み込まなければいけない。そうしないと不必要な !importantをつけなければいけなくなる。
  • SwiperはjQueryに依存しないようなので、jQueryを読み込む必要はない。
  • CSSでデザイン等を変更したい場合は、swiper-bundle.cssを参考にするといい。
  • スクロールバーは必要ないので、「<div class="swiper-scrollbar"></div>」は削除した。
  • 垂直にスライドさせたければ、JavaScript内のSwiperオプションに「direction: 'vertical',」を記述。水平のほうが良ければ削除する。

コード

HTML


<!-- CSSファイルの読み込み -->
<link rel="stylesheet" href="https://unpkg.com/swiper/swiper-bundle.min.css" />
<link rel="stylesheet" href="style.css">

<!-- JSファイルの読み込み -->
<script src="https://unpkg.com/swiper/swiper-bundle.min.js"></script>
<script src="main.js"></script>

<!-- カルーセルスライダーの表示 -->
<!-- swiperのコンテナ -->
<div class="swiper-container">
  <!-- swiperのラッパー -->
  <div class="swiper-wrapper">
    <!-- スライド画像を指定 -->
    <div class="swiper-slide"><img class="swiper-slide__img" src="slider1.jpg"></div>
    <div class="swiper-slide"><img class="swiper-slide__img" src="slider2.jpg"></div>
    <div class="swiper-slide"><img class="swiper-slide__img" src="slider3.jpg"></div>
    <div class="swiper-slide"><img class="swiper-slide__img" src="slider4.jpg"></div>
  </div>
  <!-- ページネーションが必要なら記述 -->
  <div class="swiper-pagination"></div>
  <!-- ナビゲーションボタンが必要なら記述 -->
  <div class="swiper-button-prev"></div>
  <div class="swiper-button-next"></div>
</div>

CSS


/* ----- Swiperコンテナの大きさ ----- */
.swiper-container {
  width: 100%;
  height: 100vh;
}
/* ----- Swiper画像の大きさ ----- */
.swiper-slide__img {
  width: inherit;
  height: inherit;
  object-fit: cover;
}
/* ----- ページネーションの色 ----- */
.swiper-pagination-bullet {
  background-color: #fff;
}
/* ----- 矢印の色 ----- */
.swiper-button-next,
.swiper-button-prev {
  color: #fff;
}

/* ----- ページネーションと矢印の色は、IE非対応で良いなら次のようにしてもいい ----- */
:root {
  --swiper-theme-color: #fff;
}

JavaScript


const swiper = new Swiper('.swiper-container', {
  // 各オプションパラメタ
  autoplay: true, // 自動再生するか？
  loop: true, // 繰り返し再生するか？
  speed: 1000, // 画像が切り替わる速度

  // ページネーションが必要なら記述
  pagination: {
    el: '.swiper-pagination',
  },

  // ナビゲーションボタンが必要なら記述
  navigation: {
    nextEl: '.swiper-button-next',
    prevEl: '.swiper-button-prev',
  },
});
