引数Props

デモ

コード概要

要点

  • 前回は関数によるコンポーネント化をした。今回はその関数に引数を渡す方法。「Props」と呼ばれる仕組みを試す。
  • 関数を呼び出すときに、例えば「name=""」のようにすれば、プロパティ値（属性の値）を関数に渡すことができる。属性名はおおよそ自由に決めて良いようだ。
  • 仮引数名は propsとし、関数内で「{props.name}」と書けば参照できる。関数ではなく クラス構文（class）で定義した場合は、「{this.props.name}」と書いて参照する。
  • Propsの値は、一度設定すると変更できない。Propsは読み取り専用。変更が必要な場合は、Stateを使う。

コード

HTML


<div id="root" class="demoRoot"></div>

jsx


function App(props) {
  return <p>Reactテスト：コンポーネントで{props.name}を使う</p>;
}
ReactDOM.render(
  <React.StrictMode>
    <App name="引数Props" />
  </React.StrictMode>,
  document.getElementById('root')
);
