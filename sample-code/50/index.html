押しボタン風のクリックアニメ

コード概要

要点

今回は、押しボタン風のクリックアニメーションです。ボタンをクリックすると、右下の方に少し動きます。何となく本物のボタンかのように見える動作をほどこします。ポイントは以下ようです。

  • 重要なのは、:activeです。要素に :activeをつけることでクリックしたときの動作を指定することができます。
  • ボタンを押し下げたかのように見える動作は、「transform: translate(2px, 3px);」を使用しました。
  • box-shadowで影を調整してあげることで、本物のボタンのような印象を強めます。
クリックアニメ「押しボタン風」

コード

HTML


<div class="hoverAnime">
  <a class="hoverAnime__link" href="">クリックアニメ「押しボタン風」</a>
</div>

CSS


.hoverAnime {
  text-align: center;
}
.hoverAnime__link {
  padding: .6em 3em;
  display: inline-block;
  user-select: none;
  color: #fffffc;
  background-color: #769164;
  box-shadow: 2px 2px 3px 0 rgba(0, 0, 0, .5);
  border-radius: 4px;
  transition: .3s;
}
.hoverAnime__link:hover {
  opacity: .9;
}
.hoverAnime__link:active {
  transform: translate(2px, 3px);
  box-shadow: 1px 1px 1px 0 rgba(0, 0, 0, .2),
              2px 2px 3px 0 rgba(0, 0, 0, .3) inset;
}

編集後記

あとがき

  • ボタンをクリックしたときの動作を確認しやすいように、デモ用のボタンは href属性を外してあります。
  • ページ内で使用するボタンであればかまいませんが、ページを移動してしまう場合にこのボタンを適用してしまうとあまり効果が伝わらないかもしれません。使いどころがよく分からないですね。
