【GatsbyJS入門】
パンくずリストを設置する方法

概要

要点

プラグイン gatsby-plugin-breadcrumbを使用して、GatsbyJSサイトにパンくずリストを設置する方法。以下、手順概要。（※：GatsbyJSのバージョンによっては正常に動かない可能性あり）

  1. プラグイン gatsby-plugin-breadcrumbをインストールする。
  2. gatsby-config.jsに gatsby-plugin-breadcrumbに関する設定を書く。
  3. layout.jsに gatsby-plugin-breadcrumbに必要なことを書く。
    • import { Breadcrumb } from 'gatsby-plugin-breadcrumb';
    • 引数として「location, crumbLabel」を受け取る。
    • <Breadcrumb location={location} crumbLabel={crumbLabel} />で呼び出す。
  4. CSSは「import 'gatsby-plugin-breadcrumb/gatsby-plugin-breadcrumb.css'」でインポートすれば、デフォルトの CSSがうまいことあてられて、それなりの表示になる。しかし、細かい部分で修正したければ、自分で別途 CSSファイルを作成して gatsby-browser.jsでインポートするほうがいい。
    • gatsby-browser.jsで、「import './src/styles/breadcrumb.css';」する。
    • src/styles/下に、breadcrumb.cssを作成し、gatsby-plugin-breadcrumb.cssの内容をコピーペーストする。
    • あとは好きなように変更する。

手順

コマンドライン


npm install gatsby-plugin-breadcrumb

gatsby-config.js


{
  resolve: `gatsby-plugin-breadcrumb`,
  options: {
    // defaultCrumb: optional To create a default crumb
    // see Click Tracking default crumb example below
    defaultCrumb: {
      location: {
        pathname: "/",
      },
      crumbLabel: "Home",
      crumbSeparator: " / ",
    },
    // usePathPrefix: optional, if you are using pathPrefix above
    usePathPrefix: '/blog',
  }
}

layout.js


const Layout = ({ children, location, crumbLabel }) => {
  return (
    <>
      <Header />
      <Breadcrumb location={location} crumbLabel={crumbLabel} />
      ～（中略）～
      <Footer />
    </>
  )
};

gatsby-browser.js


import './src/styles/breadcrumb.css';

src/styles/breadcrumb.css


  .breadcrumb__title {
    color: white;
    font-size: 1.5rem;
    text-decoration: none;
  }
  nav.breadcrumb {
    font-size: 1.5rem;
    font-weight: 500;
  }
  nav .breadcrumb__list {
    display: flex;
    flex-wrap: wrap;

    margin: 0;
    padding-left: 0;
  }
  nav .breadcrumb__item {
    margin: 0 0.25em;
  }

    text-decoration: none;
  }
  nav .breadcrumb__link__active {
    text-decoration: none;
  }
  nav .breadcrumb__link__disabled {
    text-decoration: none;
  }
  nav .breadcrumb__separator {
  }

編集後記

あとがき

  • GatsbyJSのバージョンが 4.0.0であれば、残念ながら次のような警告メッセージが出るかもしれない。現状のGatsbyJS最新版である 4.0に gatsby-plugin-breadcrumb 12.2.0が対応していなようだ。コマンドラインに「warn Plugin gatsby-plugin-breadcrumb is not compatible with your gatsby version 4.0.0 - It requires gatsby@^3.0.0」と出る。また、ブラウザで表示を試みると、「Unhandled Runtime Error」と出る。一度は表示されたんだけど。残念でした。
  • ドキュメントには、useBreadcrumbフックや AutoGenを使用する方法も記載されているが、どちらにせよ gatsby-plugin-breadcrumbをインストールしなければいけないようだ。パンくずリストを設置したければ、GatsbyJSのバージョンを落とすか、自作するか、対応を待つしかなさそう。
