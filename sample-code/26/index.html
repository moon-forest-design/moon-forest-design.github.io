CSSで画面をパイルアップさせる表現方法

画面を下へスクロールすると……

パイルアップしているように……

見えます。

コード概要

要点

すでにタイトルに書いた通り、画面を下へスクロールするとパイルアップ（pile up）していくように見える方法について書きます。ウィンドウ画面がスライドして積み重なっていくようなイメージです。ポイントは以下ようです。

  • 画面をパイルアップしたい要素を並べます。今回は div要素を使用しました。class名は .pileupです。
  • .pileupで重要なのは、「position: sticky;」と「top: 0;」です。これによって、ウィンドウ画面を積み重ねていくような動作がうまれます。他のプロパティは、ウィンドウにフル画面表示したり、上下左右で中央揃えする方法などです。
  • .titles--oddと .titles--evenは、色を交互につけるために使用しています。

コード

HTML


<header class="header">
  <div class="pileup">
    <h1 class="siteTitle">CSSで画面をパイルアップさせる表現方法</h1>
  </div>
  <div class="pileup">
    <p class="subTitle">画面を下へスクロールすると……</p>
  </div>
  <div class="pileup">
    <p class="subTitle">パイルアップしているように……</p>
  </div>
  <div class="pileup">
    <p class="subTitle">見えます。</p>
  </div>
</header>

CSS


.pileup {
  width: 100%;
  height: 100vh;
  display: flex;
  align-items: center;
  justify-content: center;
  position: sticky;
  top: 0;
  text-align: center;
  letter-spacing: .1em;
}
.pileup:nth-of-type(even) {
  color: #fffffc;
  background-color: #302833;
}
.pileup:nth-of-type(odd) {
  color: #302833;
  background-color: #fffffc;
}
