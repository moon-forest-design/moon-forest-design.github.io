メディア

コード概要

要点

今回は、「メディア」と呼ばれているレイアウトについて書きます。前回書いたカードと同様に、サムネイル画像・タイトル・テキストなどを一組のグループとして取り扱います。

カードとの違いは、サムネイル画像とそれに付随するタイトル・テキストは横並びになり、それら一組のグループを上詰め縦並びにレイアウトすることです。仕様は以下ようです。

  • Flexboxを使って、子要素1（サムネイル画像）と子要素2（タイトル・テキスト）を横並びにし、、2カラム表示にしています。
  • 各子要素は、flexプロパティ（flex-grow、flex-shrink、flex-basis）を使って幅の大きさを分配しています。
  • レスポンシブ対応しています。スマートフォンで見た場合はすべて 1カラム表示になります。
ストロベリームースケーキ

苺（いちご）ソースをふわふわのスポンジではさみ、その上にぷるっぷるの苺（いちご）のムースをたっぷりのせたケーキです。

ストロベリータルト

さくさくの生地の上にたっぷりの苺（いちご）ソースと新鮮な生の苺（いちご）をふんだんに使用したタルトです。

CSS


.media {
  display: flex;
  margin-bottom: 2em;
}
.mediaThumbnail {
  flex: 0 1 30%;
}
.mediaThumbnail__img {
  width: 100%;
  vertical-align: top;
  border-radius: 2px;
}
.mediaContent {
  flex: 1;
  padding: 0 2em;
}
.mediaTitle {
  font-size: 1.8rem;
  font-weight: bold;
  margin-bottom: .5em;
}
.mediaTitle__link {
  text-decoration: none;
  color: inherit;
}
@media screen and (max-width: 767px) { /* スマホ用 */

    flex-direction: column;
  }

    margin-bottom: 1em;
  }

  }
  .mediaTitle {

  }
