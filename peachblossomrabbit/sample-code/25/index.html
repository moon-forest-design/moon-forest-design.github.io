コード概要

要点

今回は、上部に表示するナビゲーション・メニューを固定する方法について書きます。positionに stickyを指定することで、ウィンドウ画面の上辺に張り付くような形になります。ポイントは以下ようです。

  • nav要素に対して、position: sticky;を指定します。一般的に nav要素は、header要素内に入れることが多いと思いますが、stickyが効くように、nav要素は header要素の外に出します。スクロールしたときに、兄弟要素がなければ stickyが効かないためです。

コード

HTML


<header class="header">
  <h1 class="siteTitle">stickyでメニューを固定する方法</h1>
</header>

<nav class="nav">
  <ul class="navList">
    <li class="navList__item"><a class="navList__link" href="">ホーム</a></li>
    <li class="navList__item"><a class="navList__link" href="#summary">コード概要</a></li>
    <li class="navList__item"><a class="navList__link" href="#relate">関連記事</a></li>
    <li class="navList__item"><a class="navList__link" href="#note">編集後記</a></li>
  </ul>
</nav>

CSS


.nav {
  position: sticky;
  top: 0;
  height: 60px;
  color: #fffffc;
  background-color: #302833;
}
.navList {
  display: flex;
  justify-content: space-between;
  align-items: center;
  height: 100%;
  list-style-type: none;
}
.navList__item {
  width: 100%;
}
.navList__link {
  display: block;
  text-align: center;
  text-decoration: none;
  color: inherit;
}

編集後記

あとがき

  • 前回の「position: sticky;」を使って、ヘッダーを固定する方法とやっていることは同じですが、stickyをあてる要素がウィンドウ画面の途中にあると、より sticky（張り付く）感が出ます。「position: fixed;」を使った固定との違いが分かり良いように思います。
