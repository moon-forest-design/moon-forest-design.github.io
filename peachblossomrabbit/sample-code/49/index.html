サムネイル画像を拡大表示するホバーアニメ

コード概要

要点

今回は、サムネイル画像を拡大表示するホバーアニメーションです。サムネイル画像の上にマウスポインターをのせると、画像が少しだけ拡大して表示されます。ポイントは以下ようです。

  • 「transform: scale(1.2);」することで、サムネイル画像を 1.2倍に拡大します。
  • 画像の親要素に「overflow: hidden;」しておくことで、サムネイル画像がはみ出した部分は表示されません。
  • アニメーションの時間は、0.7秒くらいが良さそうです。（transition: .7s;）

コード

HTML


  <ul class="hoverAnime">
  <li class="hoverAnime__item">
    <a class="hoverAnime__link" href="kids1.jpg" target="_blank" rel="noopener noreferrer">
      <img class="hoverAnime__img" src="kids1.jpg">
    </a>
  </li>
  <li class="hoverAnime__item">
    <a class="hoverAnime__link" href="kids2.jpg" target="_blank" rel="noopener noreferrer">
      <img class="hoverAnime__img" src="kids2.jpg">
    </a>
  </li>
  <li class="hoverAnime__item">
    <a class="hoverAnime__link" href="kids3.jpg" target="_blank" rel="noopener noreferrer">
      <img class="hoverAnime__img" src="kids3.jpg">
    </a>
  </li>
</ul>

CSS


.hoverAnime {
  margin: 70px 0 100px;
  display: flex;
  justify-content: space-between;
  flex-wrap: wrap;
  text-align: center;
  list-style-type: none;
}
.hoverAnime__item {
  width: calc(100% / 3 - 20px);
  margin-bottom: 30px;
  overflow: hidden;
}
.hoverAnime__img {
  width: 100%;
  transition: .7s;
}
.hoverAnime__img:hover {
  transform: scale(1.2);
}
