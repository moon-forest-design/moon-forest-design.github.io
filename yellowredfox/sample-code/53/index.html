要点

  • Reactで開発を行いたい場合は次のような方法がある。手間なく容易にできる順に並べた。1、2はお試し。3、4が本格的な開発環境。
    1. CDNを利用する。例えば unpkgなどを使えば、いつものローカル環境で試すことができるので気楽にはじめられる。サーバ環境が必要であれば、Live Serverなどを入れるといい。
    2. Code Playgroundを利用する。Web上で気軽にコードを試すことができるので便利。しかし、慣れたテキストエディタのように打ち込むことができないので小さなストレスがたまり続ける。また、アカウント登録が必要な場合もあるので、面倒くさいと感じるかもしれない。JSFiddle、CodeSandbox、codePenなど。
    3. Create React App利用する。Node.jsを導入し、Create React Appをインストールするだけで OK。
    4. モジュールバンドラを利用する。webpackなど。
  • その他、状況に応じて利用することで捗るツール。
    • サーバ側で構築しておく Webサイトを制作したい場合。Next.jsなど。
    • 静的なコンテンツ中心の webサイトを制作したい場合。Gatsbyなど。
    • JavaScriptにはないモジュール機能が使いたい場合。Browserifyなど。
    • JavaScriptを高速化したい場合。Prepackなど。
  • 本格的に開発するときに求められる要件。
    • JSXおよびモダンなJavaScriptコードの変換。
    • コンポーネント間の依存関係の管理。
    • 本番用ファイルへの最適化。
