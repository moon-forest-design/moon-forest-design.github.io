React Hooks入門
「関数によるいいねボタン」

コード概要

要点

  • React公式の「既存のウェブサイトに React を追加する」で紹介されている いいねボタン。Class Componentで書かれているので、Function Componentで書き直した。
  • 状態変数に liked、状態変更関数に clickLikedを指定し、useState()を設定。初期値は false。
  • ボタンがクリックされると、状態変更関数で状態変数の値を trueへ変更。
  • 状態変数の値が trueであれば、「You liked this !」を returnする。ボタンに関する要素は returnされないので表示されない。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  const LikeButton = () => {
    const [liked, clickLiked] = React.useState(false);

    if (liked) {
      return 'You liked this !';
    }

    return (
      <button onClick={() => clickLiked(true)}>
        Like
      </button>
    );

  };

  const reactRender = () => ReactDOM.render(
    <React.StrictMode>
      <LikeButton />
    </React.StrictMode>,
    document.getElementById('root')
  );
  reactRender();
</script>

あとがき
