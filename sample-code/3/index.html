ふわっと表示する方法

コード概要

要点

今回は、ページをスクロールしていくと、各コンテンツが少し下の位置からふわっと表れるようにする方法です。jQueryとCSSアニメーションを利用して自作しました。仕様は以下ようです。

  • 事前に、ふわっと表示したい領域（ここでは、.contents及び .content）を、CSSで透過にしておく。また、CSSの translate()を使って、100pxほど下にずらしておく。
  • ユーザーがページをスクロールし、ふわっと表示したい領域（.contents あるいは .content）に近づいたら jQueryを使いふわっと表示させる。100pxのズレも戻す。
  • リロードボタンが押された場合もふわっと表示する。

コード

HTML


<section class="contents">
 <section class="content">
  ～（コンテンツ）～
 </section>
</section>

CSS


.contents,
.content {
 opacity: 0;
 transform: translate(0, 100px);
 transition: all 1.5s;
}
.contents.visible,
.content.visible {
 opacity: 1;
 transform: translate(0, 0);
}

jQuery


$(function() {
 // コンテンツの近くにスクロールされたら　ふんわり表示する
 $(window).scroll(function() {
  const $winHeight = $(window).height(); // ウィンドウ画面の高さを取得
  $(".contents, .content").each(function() {
   const $scrollTop = $(window).scrollTop(); // スクロールした現在のtopの位置を取得
   const $scrollBottom = $scrollTop + $winHeight; // スクロールした現在のtopの位置にウィンドウ画面の高さを加算してbottomの位置を算出
   const $offsetTop = $(this).offset().top; // コンテンツの位置を取得
   if ( $scrollBottom > $offsetTop) { // ふんわり表示させたいコンテンツにスクロールが及べば表示する
    $(this).addClass("visible");
   }
  });
 });
 jQuery(window).scroll(); // 中途半端な位置でリロードされたときも、ふんわり表示する
});
