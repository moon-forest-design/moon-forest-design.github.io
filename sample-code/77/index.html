React入門
「PropTypes」

コード概要

要点

  • Reactには、型をチェックするための「PropTypes」というライブラリがあった。しかし、Reactのバージョン16.0.0以降から Reactパッケージから廃止され、外部パッケージに切り離されたそうだ。おそらく、バージョン 17で試したので warningエラーが出なかったのだろう。型チェックを行いたければ、TypeScriptを使うべし。
  • Reactには、型をチェックするためのライブラリがある。例えば、Stringなのか Numberなのか、あるいは他の予想外のおかしなデータが混入していないかを開発時にチェックするためだ。型をチェックするためのライブラリは「PropTypes」と呼ばれている。
  • 試しに 開発用の Reactと prop-types.min.jsを CDNで読み込む形にした。
  • 数値データを受け取りたいところ {this.props.num}に、あえて文字列 "Sara"を渡した。しかし、コンソールに warningエラーが出るはずなのだが、なぜか出なかった。React Developer Toolsで確認してもエラーなし。開発モードにはなっているので、コードで何か足りないのかもしれない。保留。
  • また、何かしらのトラブルで期待する値が入らない場合のためにデフォルト値を設定することができる。

コード

HTML


<!-- 開発用のReactを使用 -->
<script crossorigin src="https://unpkg.com/react@17/umd/react.development.js"></script>
<script crossorigin src="https://unpkg.com/react-dom@17/umd/react-dom.development.js"></script>
<!-- PropTypesライブラリを使用 -->
<script crossorigin src="https://unpkg.com/prop-types@15.7.2/prop-types.min.js"></script>
<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>

<!-- 数値データを受け取りたいところにあえて文字列を渡している例 -->
<div id="root" class="reactRoot"></div>
<script type="text/babel">
  class NumList extends React.Component {
    render() {
      return (
        <div>No. {this.props.num}</div>
      );
    }
  }
  // PropTypesライブラリによる型チェックに必要な設定　数値であることをチェックしたい
  NumList.propTypes = {
    num: PropTypes.number
  };
  const render = () => ReactDOM.render(
    // 試しにあえて文字列を渡す
    <React.StrictMode>
      <NumList num="Sara" />
    </React.StrictMode>,
    document.getElementById('root')
  );
  render();
</script>
