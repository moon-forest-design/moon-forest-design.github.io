タイトルを上下左右で中央揃えにする方法

コード概要

要点

今回は、タイトルを上下左右で中央揃えにする方法について書きます。よく知られているものを 5通りだけ書きました。サンプルとして、以下の見出し部分を上下左右で中央揃えにしてあります。

  1. paddingと text-alignを使う方法
  2. line-heightと text-alignを使う方法
  3. flexを使う方法
  4. gridを使う方法
  5. table-cellを使う方法

サンプルとして、以下の見出し部分を上下左右で中央揃えにしてあります。背景色が桃色の部分です。見出しの下にそれぞれの説明を入れています。

1：paddingと text-alignを使う方法

  • 1つ目は、paddingと text-alignプロパティを使う方法です。
  • 上下は paddingで中央揃えにし、左右は text-alignで中央揃えにしています。
  • 親要素に heightを指定してしまうとうまくいきません。上下でズレます。

2：line-heightとtext-alignを使う方法

  • 2つ目は、line-heightと text-alignプロパティを使う方法です。
  • 上下は line-heightで中央揃えにし、左右は text-alignで中央揃えにしています。
  • 親要素の heightの値を変更するときは、忘れずに line-heightの値も変更する必要があります。少し手間です。

3：flexを使う方法

  • 3つ目は、flex（display: flex;）を使う方法です。
  • 上下は align-itemsプロパティで中央揃えにし、左右は justify-contentプロパティで中央揃えにしています。
  • 親要素に heightが指定されていてもいなくてもどちらでも大丈夫です。
  • IE11にも対応しています。

4：gridを使う方法

  • 4つ目は、grid（display: grid;）を使う方法です。
  • 上下左右を place-itemsプロパティで中央揃えにしています。
  • 親要素に heightが指定されていてもいなくてもどちらでも大丈夫です。
  • 残念ながら、IE11に対応していません。

5：table-cellを使う方法

  • 5つ目は、table（display: table-cell;）を使う方法です。
  • 上下は 子要素に「display: table-cell;」「vertical-align: middle;」することで中央揃えにし、左右は 親要素に「display: inline-block;」、及び大親要素で「text-align: center;」することにより中央揃えしています。
  • 親要素に heightを指定しなければいけません。子要素は heightを継承する必要があります。
  • HTML側で、div要素（.header__inner）を増やす必要があります。手間が増えます。
  • IE11にも対応しています。

コード

HTML


<-- 1～4用 -->
<header class="header">
  <h1 class="title">タイトル</h1>
</header>

<-- 5：table-cell用 -->
<header class="header">
  <div class="header__inner">
    <h1 class="title">タイトル</h1>
  </div>
</header>

CSS


/* 1：paddingと text-alignを使う方法 */
.title {
  padding: 2em 1em;
  text-align: center;
}

/* 2：line-heightと text-alignを使う方法 */
.header {
  height: 120px;
}
.title {
  line-height: 120px;
  text-align: center;
}

/* 3：flexを使う方法 */
.header {
  display: flex;
  align-items: center;
  justify-content: center;
}

/* 4：gridを使う方法 */
.header {
  display: grid;
  place-items: center;
}

/* 5：tableを使う方法 */
.header {
  text-align: center;
}
.header__inner {
  display: inline-block;
  height: inherit;
}
.title {
  display: table-cell;
  height: inherit;
  vertical-align: middle;
}

編集後記

あとがき

どれが優れているということはなく、今のところ状況に応じて使い分けています。今後さらに、いろいろと良い方法が出てくると思うので、どのようにするのが良いものか見直しつつ書き加えていきます。

  • 親要素に heightを指定しなくて良いデザインレイアウトであれば、「1：paddingと text-alignを使う方法」で済みます。
  • 親要素に heightを指定する場合は、「3：flexを使う方法」を使います。
  • 今後は、「4：gridを使う方法」が増えていくかもしれません。
