コード概要

要点

今回は、スムーズスクロール機能です。scroll-behaviorプロパティを使えば、CSSだけでスムーズにスクロールができてしまうとのことなので試してみました。仕様は以下ようです。

  • 事前に、HTMLファイル内の各 h2見出しに、それぞれ idをつけておきます。上部メニューには、各 h2見出しへページ内移動するリンクをつけてあります。
  • html要素に「scroll-behavior: smooth;」をあてます。

コード

HTML


<h2 id="summary">コード概要</h2>
（～中略～）
<h2 id="relate">関連記事</h2>
（～中略～）
<h2 id="note">編集後記</h2>

CSS


html {
  scroll-behavior: smooth;
}

編集後記

あとがき

  • 「scroll-behavior: smooth;」は、使えると簡便でありがたいのだけれど、ブラウザ対応がまだ足りていません。IEはしょうがないとしても、Safariもダメとなると現状においては使い難い感じがいたします。
  • ページ内検索時に「ゆっくりとスクロールされる」という不具合もあるようです。改善策はあるので問題はないのでしょうけれど、ただ、ブラウザ対応の問題も含め、そこまでしてという気持ちになってしまいます。もう少し様子を見たい機能です。
