月の森デザイン

コード概要

要点

ヘッダーをウィンドウ画面いっぱいに表示して、タイトルを上下左右中央揃えにする方法を書き残しておきます。CSSのポイントは以下ようです。

  • html要素に、「height: 100%;」を指定する。
  • html要素のheightが継承されるように、タイトルの親要素まで「height: inherit;」を指定する。（例：body要素、header要素、div要素……など）
  • タイトルの親要素に、flexで上下左右中央揃えを指定する。

コード

HTML


<header class="header">
　<div class="header__container">
　　<h1 class="siteTitle" data-subtitle="ホームページ制作の">月の森デザイン</h1>
　</div>
</header>

CSS


html {
　height: 100%;
}
body {
　height: inherit;
}
.header {
　height: inherit;
}
.header__container {
　height: inherit;
　display: flex;
　align-items: center;
　justify-content: center;  
}
