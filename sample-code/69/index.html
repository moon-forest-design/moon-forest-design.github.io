React入門
「関数で定義したイベント処理」

コード概要

要点

  • button要素をクリックすると、onSubmit属性に指定している関数handleSubmitを呼び出す。関数内では2つの処理を行っている。
  • 1つは、button要素をクリックしたことで発生するデフォルト処理を抑止するために、「e.preventDefault();」している。
  • 2つ目は、クリックされたことが分かるよう「You clicked submit.」とコンソール表示している。

コード

HTML


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  function Form() {
    function handleSubmit(e) {
      // button要素をクリックしたことで発生するデフォルト処理を抑止
      e.preventDefault();
      console.log('You clicked submit.');
    }

    return (
      <form onSubmit={handleSubmit}>
        <button type="submit">Submit</button>
      </form>
    );
  }

  ReactDOM.render(
    <React.StrictMode>
      <Form />
    </React.StrictMode>,
    document.getElementById('root')
  );

</script>
