space-betweenの最後の行を左寄せにする方法

コード概要

要点

Flexboxの「justify-content: space-between;」は、とても便利です。さらに「flex-wrap: wrap;」を使えば、二行になってもうまいこと……おさまらないんですよね。

例えば次のように、カラムや子要素の数によっては、最後の行が左寄せにならないのです。「こういうのもデザイン的に、有りかも？」などと言えるはずもなく。どうしたものか考える必要があります。

親要素の幅を固定にしてもかまわなければ、justify-contentに flex-startなどを指定すれば良いかもしれません。しかし、状況によっては可変にしたい場合もあるはずです。

調べてみたところ、二つの方法がみつかりました。1つは、疑似要素を使う方法。もう一つは、jQueryを使う方法です。どちらも共通するのが、空の要素を追加することで穴埋めし、きれいに整えるということ。

今後、justify-contentが進化して簡単に対応できるようになるかもしれませんが、現状においてはこれらの方法を採用するしかなさそうです。一部参考にしつつ、コードを書いてみました。仕様は以下ようです。

  • 親要素の幅は、可変でもかまいません。
  • 最後の行で不足する子要素の数だけ、空要素を追加します。

コード

HTML


<div id="parent" class="parent">
  <div class="child"></div>
  <div class="child"></div>
  <div class="child"></div>
  <div class="child"></div>
  <div class="child"></div>
  <div class="child"></div>
  <div class="child"></div>
  <div class="child"></div>
</div>

CSS


.parent {
  display: flex;
  justify-content: space-between;
  width: 540px; /* 削除可 */
  flex-wrap: wrap;
}
.child {
  width: 100px;
  height: 100px;
  margin-bottom: 10px;
  background-color: #ec6800; /* 削除可 */
}
.isEmpty {
  height: 0;
  padding-top: 0;
  padding-bottom: 0;
  margin-top: 0;
  margin-bottom: 0;
}

jQuery


$(function() {
  const $cellWidth = $('#parent .child').outerWidth(true); // 子要素の幅を取得
  const $cellCount = $('#parent .child').length; // 子要素の数を取得
  const $parentWidth = $('#parent').width(); // 親要素の幅を取得
  const $columnCount = Math.trunc( $parentWidth / $cellWidth ); // 親要素の幅に設定可能なカラム数を算出
  let $lackCell = 0; // 左寄せするために不足している子要素の数を最終的に入れる
  let $gridCount = 0; // 行列（グリッド）にした場合の子要素の総数を計算するときに使用
  while ($gridCount < $cellCount) { // 行列（グリッド）にした場合の子要素の総数を算出
    $gridCount += $columnCount;
  }
  $lackCell = $gridCount - $cellCount; // 左寄せするために不足している子要素の数を算出
  for (let $j = 0; $j < $lackCell; $j++) { // 左寄せするために不足している子要素の数だけ空要素を付け加える
    $('#parent').append('<div class="child isEmpty"></div>');
  }

});
