React Hooks入門
「関数によるtoggleボタン」

コード概要

要点

  • class構文で書かれていた Toggleボタンのコードを、関数で書き直してみた。

コード

HTML + React


  <div id="root" class="reactRoot"></div>
  <script type="text/babel">
    const Toggle = () => {
      // ステートフック
      const [isToggleOn, handleClick] = React.useState(true);
      // ON OFF切り替え（true or false）
      const toggleChange = () => !isToggleOn;
      return (
        <button onClick={() => handleClick(toggleChange)}>
          {isToggleOn ? 'ON' : 'OFF'}
        </button>
      );
    };
    ReactDOM.render(
      <React.StrictMode>
        <Toggle />
      </React.StrictMode>,
      document.getElementById('root')
    );
  </script>

編集後記

あとがき

  • toggleChangeの中身はこれくらいなら、handleClick(() => !isToggleOn)としても良いと思うけれど、分けたほうが見やすい感じがしたので外に出した。
  • ステートフック useState()はつまり、簡便な方法でクロージャのテクニックを使えるようにしてくれているのかな。近しい。
