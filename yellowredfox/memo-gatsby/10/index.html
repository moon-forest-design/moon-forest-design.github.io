CSS Modules

概要

要点

  1. src/components/下に layout.module.cssファイルを作成する。とりあえず、.containerの指定だけ記述。
  2. src/components/下の layout.jsに以下指定。
    • importする。例：import * as style from './layout.module.css';
    • div要素に className属性で指定。例：<div className={style.container}>

手順

layout.module.css


.container {
  margin: auto;
  max-width: 500px;
  font-family: sans-serif;
}

layout.js


import * as React from 'react';
import { Link } from 'gatsby';
import * as style from './layout.module.css';

const Layout = ({ pageTitle, children }) => {
  return (
    <div className={style.container}>
      <title>{pageTitle}</title>
      <nav>
        <ul>
          <li>
            <Link to="/">Home</Link>
          </li>
          <li>
            <Link to="/about">About</Link>
          </li>
        </ul>
      </nav>
      <main>
        <h1>{pageTitle}</h1>
        {children}
      </main>
    </div>
  )
};

export default Layout;

編集後記

あとがき

  • GatsbyJSサイトに styleを指定する方法はたくさんある。単に CSSファイルを importしてもいいし、CSS in JSや Tailwind CSSなどのフレームワークを使用してもかまわないとのこと。ただ、CSS Modulesを推してはいるようなので、試してみた。
  • importは、「import { container } from './layout.module.css';」という書き方もできるが、数が増えるとものすごい手間なので見合わせた。
  • 「import styles from './layout.module.css';」でも良いのではと思ったが、エラーが出るのでダメだった。
