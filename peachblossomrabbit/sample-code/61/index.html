コード概要

要点

今回も、ドロワーメニューです。前回はドロワーメニューを自作しましたが、今回は Drawer（drawer.jsというJavaScriptライブラリ）を使用します。ポイントは以下ようです。

  • htmlに関しては、Drawer公式に書いてある手順通りに進めれば問題ありません。
  • 右に配置したかったので、body要素のclass名は「drawer--right」にしました。
  • 見出し（h1要素）を追加しました。見出し含めヘッダーは固定です。
  • 読み込んだ Drawerの CSS（drawer.min.css）の設定そのままでも良い場合があるかもしれませんが、サイトの状況に応じて調整が必要になります。drawer.cssを参考にして、色や配置を少々変更しました。
  • main.jsから Drawerに関するオプション指定をいろいろとできるようですが、今回はとくに何もせずにデフォルト設定のままにしました。

コード

HTML


  <!-- drawer.jsで使用されるCSSファイルを読み込む -->
  <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/drawer/3.2.2/css/drawer.min.css">

  <!-- iScrollと drawer.jsを読み込む -->
  <script src="https://cdnjs.cloudflare.com/ajax/libs/iScroll/5.2.0/iscroll.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/drawer/3.2.2/js/drawer.min.js"></script>

<!-- drawer.jsで必要となるマークアップ例 -->
<body class="drawer drawer--right">
  <header class="header" role="banner">
    <div class="signBoard">
      <h1 class="siteTitle">ドロワーメニュー<br class="brNone">（drawer.js版）</h1>

      </button>

        <ul class="drawer-menu">

CSS


/* メニュー全体 */
.drawer-nav {
  padding-top: 1rem; /* 初期値：なし */
  background-color: #302833; /* 初期値：#fff */
}
  /* メニュー大項目 */
.drawer-brand {
  font-size: 1.6rem; /* 初期値：1.5rem */
  padding-left: 1rem; /* 初期値：.75rem */
  color: #fff; /* 初期値：#222 */
}
/* メニュー小項目 */
.drawer-menu-item {
  font-size: 1.5rem; /* 初期値：1rem */
  padding: .75rem .75rem .35rem 2.75rem; /* 初期値：.75rem */
  color: #fff; /* 初期値：#222 */
}
/* ハンバーガーボタン全体 */
.drawer-hamburger {
  width: 3rem; /* 線幅　初期値：2rem */
  padding-top: 35px; /* 位置調整　初期値：18px */
  padding-right: 1.8rem; /* 位置調整　初期値：.75rem */
}
/* ハンバーガーボタンの三本線 */
.drawer-hamburger-icon,
.drawer-hamburger-icon:after,
.drawer-hamburger-icon:before {
  background-color: #fff; /* 線色　初期値：#222 */
}
/* オーバーレイ */
.drawer-overlay {
  background-color: rgba(0, 0, 0, .7); /* 初期値：0, 0, 0, .2 */
}
