クリックイベント付きの div要素を
for文で 5つ表示する

コード概要

要点

  • div要素をクリックすると、何番目の div要素かをコンソール表示する仕様。

コード

JavaScript


for (let i = 0; i < 5; i++) {
  const demo = document.getElementById('demo');
  const div = document.createElement('div');
  div.onclick = function() {
    console.log(`This is box # ${i}`);
  }
  demo.appendChild(div);
}
