【React入門】
React.Fragment（フラグメント）

デモ

コード概要

要点

  • Reactでは複数の要素を返したい場合に、div要素などでひとつの要素にまとめる必要がある。しかし、div要素で囲んでしまうとおかしくなってしまう場合もある。例えば、複数の td要素だけを返したい場合に、div要素で囲うのはおかしい。
  • div要素が不適切かつ代替となる方法がない場合は、「React.Fragment（フラグメント）」を使うと良い。<React.Fragment>...</React.Fragment>で囲んであげると、これらの記述は出力せず無視されるとのことだ。
  • 短縮記法もある。<>...</>で囲めば同じ意味になる。ただし、フラグメントの短縮記法を認識できないツールがあるそうなので、使って良いものかは分からない。

コード

HTML


<div id="root" class="demoRoot"></div>

jsx


  function DataView() {
    return (
      <React.Fragment>
        <td>データA</td>
        <td>データB</td>
        <td>データC</td>
      </React.Fragment>
    );
  }
  ReactDOM.render(
    <React.StrictMode>
      <table>
        <tr>
          <th>見出しA</th>
          <th>見出しB</th>
          <th>見出しC</th>
        </tr>
        <tr><DataView /></tr>
      </table>
    </React.StrictMode>,
    document.getElementById('root')
  );
