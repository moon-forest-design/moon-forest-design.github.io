Layoutコンポーネント

概要

要点

  1. src/components/下に layout.jsファイルを作成する。index.jsや about.jsから引数（props）を受け取り、引数を使って生成した要素群を returnする関数。
  2. src/pages/下の index.jsから <Layout>を使って呼び出す。pageTitleのように直接明示して引数を渡す形、<Layout>～</Layout>の中に記述するだけで引数 childrenとして渡される形、2つの方法がある。

手順

layout.js


import * as React from 'react'
import { Link } from 'gatsby'

const Layout = ({ pageTitle, children }) => {
  return (
    <div>
      <title>{pageTitle}</title>
      <nav>
        <ul>
          <li><Link to="/">Home</Link></li>
          <li><Link to="/about">About</Link></li>
        </ul>
      </nav>
      <main>
        <h1>{pageTitle}</h1>
        {children}
      </main>
    </div>
  )
}

export default Layout

index.js


import * as React from 'react'
import Layout from '../components/layout'

const IndexPage = () => {
  return (
    <Layout pageTitle="Home Page">
      <p>I'm making this by following the Gatsby Tutorial.</p>
    </Layout>
  )
}

export default IndexPage

編集後記

あとがき

  • GatsbyJSでは、各ページの重複する部分を外部ファイルに書き出しておいて importすることができる。これらは、ビルディングブロックコンポーネントと呼ばれている。今回は、Layoutコンポーネントを試した。headerや footerも同様の措置を行う。
