React Hooks入門
「名前入力欄」

コード概要

要点

  • React公式の「フォーム」で紹介されているコード。Class Componentで書かれているものを、Function Componentで書き直した。
  • 入力欄に名前を入力し、サブミットボタンを押すとポップアップメッセージが表示される仕様。やっていることは、React入門「関数によるTodoリスト」とほぼ同じ。入力されたデータをストックしないだけ。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  const NameForm = () => {
    const [value, setvalue] = React.useState("");

    const handleChange = (event) => {
      setvalue(event.target.value);
    };

    const handleSubmit = (event) => {
      event.preventDefault();
      alert('A name was submitted: ' + value);
      setvalue("");
    }

    return (
      <form onSubmit={handleSubmit}>
        <label htmlFor="name">名前：</label>
        <input id="name" type="text" value={value} onChange={handleChange} />
        <input type="submit" value="送信（テスト）" />
      </form>
    );

  };
  ReactDOM.render(
    <React.StrictMode>
      <NameForm />
    </React.StrictMode>,
    document.getElementById('root')
  );
</script>
