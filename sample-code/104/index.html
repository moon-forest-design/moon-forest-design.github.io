React入門
「クラス構文によるjsonplaceholderのAPI」

コード概要

要点

  • 「React Component with Fetch API」で紹介されているコード。jsonplaceholderの APIを利用してデータを取得し、表示する仕様。取得するデータは photoに変更した。また、「?」をつけて、1件（id=1）のみにした。うまくいけば、サイズ 150 x 150のグリーンの画像が表示されるはず。
  • jsonplaceholderは、Fetch APIのテストなどに利用できるサイトとのこと。アカウント登録は不要。「Fetch APIをちょっと試したいな」というニーズに応えてくれる。データはダミー。
  • 以前試した「WikipediaのAPI」には例外処理がなかったが、今回のコードには catch()あり。しかし、コンソールにエラーをはくわけでもなく、とくに何を表示するでもない仕様となっているようだ。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  const Photos = ({ photo }) => {
    return (
      <figure className="thumbnail">
        <img src={photo.thumbnailUrl} alt={photo.title} className="thumbnail__img" />
        <figcaption className="thumbnail__title">{photo.title}</figcaption>
      </figure>
    );
  };

  class App extends React.Component {
    state = {
      photos: [],
      isLoading: true,
      errors: null
    };

    fetchPhotos() {
      fetch(`https://jsonplaceholder.typicode.com/photos?id=1`)
        .then(response => response.json())
        .then(data =>
          this.setState({
            photos: data,
            isLoading: false
          })
        )
        .catch(error => this.setState({ error, isLoading: false }));
    }

    componentDidMount() {
      this.fetchPhotos();
    }

    render() {
      const { isLoading, photos } = this.state;
      return (
        <section className="section">
          <div className="container">
            {!isLoading ? (
              photos.map(photo => {
                return <Photos key={photo.id} photo={photo} />;
              })
            ) : (
              <p>Loading...</p>
            )}
          </div>
        </section>
      );
    }
  }

  ReactDOM.render(<App />, document.getElementById("root"));

</script>
