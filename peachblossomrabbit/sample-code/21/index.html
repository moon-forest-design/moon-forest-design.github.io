background-attachmentで
背景画像を固定する方法

コード概要

要点

今回は、CSSの background-attachmentプロパティを使って、背景画像を固定する方法について書きます。backgroundで画面いっぱいに表示した画像を固定することで、スクロールすると画像の下に位置するコンテンツ部分（main要素）が少しずつかぶっていくような表示になります。ポイントは以下ようです。

  • 空の div要素を用意し、CSSの backgroundプロパティで背景画像（ヒーローイメージ）を指定する。
  • 背景画像に、background-attachment: fixed;を指定する。

コード

HTML


<div class="mainVisual">
  <!-- CSSのbackgroundで画像を背景表示する領域 -->
</div>

CSS


.mainVisual {
  width: 100%;
  height: 100vh;
  margin-bottom: 10em;
  background: center/cover url("mainVisual.jpg");
  background-attachment: fixed;
}
