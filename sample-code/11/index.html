レイアウトに役立つ max-widthの使い方

コード概要

要点

今回は、CSSの max-widthのお話です。例えば仮に、画面幅が 1280pxのモニタがあるとします。CSSでテキスト部分を画面幅の 90%（width: 90%;）に指定した場合、次のようにおさまり良く表示されるのでとくに問題はありません。

このようなときは max-widthを指定しておくと、うまいこと解決できます。例えば、最大で 900pxまで表示する形に幅を指定しておきたければ、「max-width: 900px;」とすれば良いのです。

max-widthを使えば、「最大（max）でここまで！」という限界幅が指定できるので、どんなに画面幅が大きくなっても問題がなくなります。max-widthは、状況に応じて widthを上書きしてくれる役立つプロパティです。

あとがき

近年、パソコンのモニタ画面は、どんどん大きくなる傾向にあります。widthに %だけ指定すると、意図しない表示状態になる可能性が増えてしまいます。max-widthをうまいこと使えば、レスポンシブ対応で役立ちます。
