ローディング待ちを表示する方法

コード概要

要点

今回は、ローディング待ちを表示する方法について書きます。ローディング待ちとは、画面上に表示されるべきはずの要素がすべて表示されず、閲覧者が待っている状態のことを言います。そのような待ち状態のときに何も表示されないと、何が起きているのか分かりにくいため、閲覧者を不安にさせてしまう可能性があります。

閲覧者に不安感を与えないためには、「待ち状態」であることをお伝えするほうが親切です。一般的には、待ち状態であることを伝えために、くるくる回るような画像を使用することが多いようです。

何の話だか分からない場合は、ブラウザのリロードボタンを押していただくと、デモ用のローディング待ち画像が改めて表示されるはずです。お試しあれ。

制作する上でのポイントは、以下ようです。

  • ローディング待ちの画像（loading.gif）を用意します。
  • 画像（loading.gif）は、CSSで固定配置にして、さらに上下左右中央揃えにします。
  • 要素がすべて表示（ロード）されたら、jQueryを使って、非表示に（フェードアウト）しています。

コード

HTML


<div class="loadingWait">
  <img class="loadingWait__img" src="loading.gif">
</div>

CSS


.loadingWait {
  width: 100px;
  height: 100px;
  position: fixed;
  top: calc(50% - 50px);
  right: 0;
  left: 0;
  margin: 0 auto;
}
.loadingWait__img {
  width: 100%;
}

jQuery


$(function() {
  $(".loadingWait__img").fadeOut("slow");
});

編集後記

あとがき

  • デモ用では、fadeOut(3000)にしてあります。長めに設定しないと一瞬で消えてしまい、デモの意味がなくなるためです。"slow"を指定するか、あるいは指定なしが良いかは、そのサイトによります。
  • フェードアウトではなく、ローディング待ちの画像をさっさと消してしまいたいという方は、fadeOut()ではなく、hide()を使うと良いでしょう。
