React入門
「Hello React!」

コード概要

要点

  • 簡便にReactを試すだけなら、関連するライブラリ群を CDNで読み込んで、HTMLファイル内にReactのコードを書くのがもっとも手間がなく Reactだけに集中できて良い。しかし、「Reactのコードを HTMLファイル内に書きたくない。HTMLファイル外に書き出したい」と思ったならば、何かしらの方法でサーバを起てる必要がある。
  • Node.jsにて、Create React Appを入れても良いが、「そこまでではない」ということであれば、VSCodeの拡張 Live Serverを使うと良いかもしれない。今回は Live Serverを入れて、HTMLファイル内にReactのコードを、外部ファイル index.jsxに書き出した。問題なければ、「Hello React!」と表示される。

コード

HTML


<!-- reactを使用する上で必要なものをCDNで読み込む -->
<script src="https://unpkg.com/react@17/umd/react.development.js"></script>
<script src="https://unpkg.com/react-dom@17/umd/react-dom.development.js"></script>
<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>

<!-- Reactを書き出すための div要素 -->
<div id="root" class="reactRoot"></div>

<!-- Reactのコードを外部ファイルから読み込む -->
<script type="text/babel" src="index.jsx"></script>

JavaScript（jsxファイル）


ReactDOM.render(
  <p>Hello React!</p>,
  document.getElementById('root')
);
