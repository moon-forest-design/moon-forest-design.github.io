「JSX」

コード概要

要点

  • 「JSX」とは？
    • 「JSX」は、JavaScriptの拡張言語。XMLの文法に従って記述するルールがある。
    • Reactで要素を使用する場合、React.createElement()という形式で記述する必要があるが、JSXを使えば一般的な HTML要素のような形で記述できるようになる。
    • 外部ファイル化する場合は、拡張子を「.jsx」とする。
  • JSXの主な書き方。
    • ()かっこ。例えば、return ();にすれば、複数行の要素を書いてもいい。ただし、returnは単一要素にしないとエラーになるので、div要素で囲む。
    • {}かっこ。JSX内で JavaScriptのコードを書きたい場合に使う。例えば、HTML内で変数を使いたいときに、constや letで宣言した値を、{変数名}と書くことで参照できるようになる。また、関数を呼び出すこともできる。
    • 予約語とバッティングして使えない属性名がある。例えば、class属性は classNameと書く。for属性は htmlForと書く。value属性は defaultValueと書く。checked属性は defaultCheckedと書く。-（ハイフン）でつながず、キャメルケース。
    • CSSをあてるときは、style={{}}として、JSON形式で書く。
    • コメントは、{/* コメント */}のように書く。
    • 終了タグ（閉じタグ）のない要素は、 / をつける。例えば、<App />のように書く。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  function App() {
    const prefix = "Reactテスト：";
    return (
      <div className="reactText">
        <p style={{ fontWeight: 'bold' }}>{prefix}JSX</p>
        <p>JSXを使って書いた</p>
      </div>
    );
  }
  ReactDOM.render(
    <React.StrictMode>
      <App />
    </React.StrictMode>,
    document.getElementById('root')
  );
</script>
