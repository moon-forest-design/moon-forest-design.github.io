ボタンを透過するホバーアニメ

コード概要

要点

今回は、ボタンを透過するホバーアニメーションです。ポイントは以下ようです。

  • a要素をボタンに仕立てます。「padding: .6em 3em;」で余白を調整し、ボタンの大きさをきめます。また、影（box-shadow）をつけたり、角丸（border-radius）にすることで、よりボタンらしく見えるようにします。
  • ホバーされたら、透過（opacity: .8;）します。
  • 一瞬でパッと変わっても良いですが、じんわり変化させるために transitionも指定しました。アニメーションの時間は、0.5sです。（transition: .5s;）
ホバーアニメ「マウスをのせると透過します」

コード

HTML


<div class="hoverAnime">
  <a class="hoverAnime__link" href="">ホバーアニメ「マウスをのせると透過します」</a>
</div>

CSS


.hoverAnime {
  text-align: center;
}
.hoverAnime__link {
  padding: .6em 3em;
  color: #fffffc;
  background-color: #69821b;
  box-shadow: 2px 2px 3px 0 rgba(0, 0, 0, .5);
  border-radius: 4px;
  transition: .5s;
}
.hoverAnime__link:hover {
  opacity: .8;
}

編集後記

あとがき

  • ボタンを透過するホバーアニメーションは、うっすらと透過させることで、押下（クリック）できることを気づきやすくさせるメリットがあります。定番化しているため、他の凝ったホバーアニメーションにするよりも、クリック率は高まるかもしれません。
