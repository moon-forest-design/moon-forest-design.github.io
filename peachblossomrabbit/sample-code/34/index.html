高さを固定して
Googleマップを表示する方法

コード概要

要点

今回は、Googleマップを表示する方法について書きます。横幅をウィンドウ画面いっぱい、高さはそのままに表示したい場合です。導入の作業手順は以下ようです。

  1. Googleマップへアクセスする。
  2. 住所を入力し検索する。
  3. 「共有」ボタンをクリックする。
  4. 「地図を埋め込む」タブをクリックする。
  5. 「HTMLをコピー」をクリックする。
  6. HTMLファイルの表示したい箇所に貼り付ける。
  7. CSSで widthを 100%にする。

コード

HTML


<div class="googleMap">
  <iframe src="～（省略）～" width="600" height="450" style="border:0;" allowfullscreen="" loading="lazy"></iframe>
</div>

CSS


.googleMap iframe {
  width: 100%;
}

編集後記

あとがき

  • 今回は高さを固定していますが、案件によって（あるいは多くの場合？）は縦横比を維持して表示させることもあるようです。一般的には、positionや paddingを駆使して実現させているようです。しかし面倒くさい印象があります。それなら「calc()は使えないだろうか」と思いついて試してみました。例えば、600:450（4:3）なら、「height: calc(100vw * 0.75);」で良さそうです。ところが、vwという単位はスクロールバーの値も含むそうです。微妙に歪むのかな。残念でした。
  • Googleマップの縦横比を維持して表示する方法を書きました。JavaScriptを使用しています。
