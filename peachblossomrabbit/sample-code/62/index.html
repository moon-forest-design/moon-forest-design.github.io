CSSで画像を正方形に
トリミングする方法

コード概要

要点

今回は、CSSで画像を正方形にトリミングする方法です。ポイントは以下ようです。

  • 画像の親要素（.mediaThumb__link）に、画像サイズをpxで指定します。今回は、親要素が a要素なので「display: block;」もしてあります。
  • 画像要素（.mediaThumb__img）は、widthとheightをどちらも 100%にし、「object-fit: cover;」します。

CSS


.media {
  display: flex;
  margin-bottom: 2em;
}
.mediaThumb {
  flex: 0 1 300px;
}
.mediaThumb__link {
  display: block;
  width: 300px;
  height: 300px;
}
.mediaThumb__img {
  width: 100%;
  height: 100%;
  object-fit: cover;
  border-radius: 2px;
}
.mediaContent {
  flex: 1;
  padding: 0 2em;
}
.mediaTitle {
  font-size: 1.8rem;
  font-weight: bold;
  margin-bottom: .5em;
}
.mediaTitle__link {
  text-decoration: none;
  color: inherit;
}
@media screen and (max-width: 767px) { /* スマホ用 */
  .media {
    flex-direction: column;
  }
  .mediaThumb {
    margin-bottom: 1em;
  }
  .mediaThumb__link {
    margin: 0 auto;
  }
  .mediaContent {
    padding: 0;
  }
  .mediaTitle {
    text-align: center;
  }
}

あとがき

  • 「media」や「card」と呼ばれるレイアウトでサムネイル画像を表示する場合、正方形にしたいことがあります。Photoshopなどの画像編集ソフトでトリミングしてもかまいませんが、状況によってはいたしかねることもあります。そんなときに有用な方法です。
