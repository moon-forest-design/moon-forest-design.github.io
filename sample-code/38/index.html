Slickの使い方

コード概要

要点

今回は、カルーセルスライダー「Slick」の使い方です。ポイントは以下ようです。

  • Slick本体はダウンロードしてもいいし、CDNを利用してもどちらでも構わない。今回は、手間が少ないので CDNを利用した。
  • slick.cssと slick-theme.cssの後に style.cssを読み込まなければいけない。そうしないと !importantをつけなければいけなくなる。無意味。順番をまもりましょう。
  • jquery.min.jsの後に、slick.min.jsを読み込まなければいけない。Slickは jQueryライブラリなので順番を間違うと動かない。
  • CSSでデザイン等を変更したい場合は、slick-theme.cssを見ればいい。slick-theme.cssで指定されている class名に対して、style.cssで追加・変更する形となる。
  • .slick-prevの「z-index: 100;」は指定したほうがいい。他の場所で position系（static以外）の処理を使っていると、階層の問題で表示されなくなることがある。

コード

HTML


<!-- CSSファイルの読み込み -->
<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick.css"/>
<link rel="stylesheet" type="text/css" href="https://cdnjs.cloudflare.com/ajax/libs/slick-carousel/1.8.1/slick-theme.css">
<link rel="stylesheet" href="style.css">

<!-- JSファイルの読み込み -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<script type="text/javascript" src="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick.min.js"></script>
<script src="main.js"></script>

<!-- カルーセルスライダーの表示 -->
<ul class="slider">
  <li class="slider__item"><img class="slider__img" src="slider1.jpg"></li>
  <li class="slider__item"><img class="slider__img" src="slider2.jpg"></li>
  <li class="slider__item"><img class="slider__img" src="slider3.jpg"></li>
  <li class="slider__item"><img class="slider__img" src="slider4.jpg"></li>
</ul>

CSS


/* ----- リストマーカー非表示 ----- */
.slider {
  list-style-type: none;
}
/* ----- 矢印の位置 ----- */
.slick-next {
  right: 45px;
  z-index: 99;
}
.slick-prev {
  left: 20px;
  z-index: 100;
}
/* ----- 矢印の大きさ ----- */
.slick-prev:before,
.slick-next:before {
  font-size: 50px;
}
/* ----- 画像の大きさ ----- */
.slider__img {
  width: 100%;
  height: 100vh;
  object-fit: cover;
}
/* ----- ページネーションの位置 ----- */
.slick-dots {
  bottom: 25px;
}
/* ----- ページネーションの色 ----- */
.slick-dots li.slick-active button:before {
  color: #fff;
}

jQuery


$(function() {
  $('.slider').slick({
    autoplay: true, // 自動再生するか？
    autoplaySpeed: 3000, // 自動再生時の間隔ms
    speed: 1000, // 画像が切り替わる速度
    pauseOnHover: false, // 画像にマウスオーバしたときに停止するか？
    dots: true, // ページネーションを表示するか？
  });
});
