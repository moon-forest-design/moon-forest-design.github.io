Gridレイアウト入門その1

コード概要

要点

今回は、Gridレイアウトです。サムネイル画像を並べるだけの簡単なレイアウトを試してみます。ポイントは以下ようです。

  • 画像は div要素（.grid__item）に入れます。それらの div要素（.grid__item）を、親 div要素（.grid）で囲みます。親要素となる div要素（.grid）に対して「display: grid;」を指定することで Gridコンテナ化します。
  • 3列に並べたいので、「grid-template-columns」プロパティを使います。「grid-template-columns: 1fr 1fr 1fr;」と指定するだけで 3列に並びます。単位 frは fraction（分数）の略称です。ただ「grid-template-columns: repeat(3, 1fr);」と書くほうがスマートです。一般的には、repeat()を使って書きます。
  • スマートフォンサイズでは、2列にするためにメディアクエリで「grid-template-columns: 1fr;」と指定しました。
  • タブレットサイズでは、2列にするためにメディアクエリで「grid-template-columns: repeat(2, 1fr);」と指定しました。
  • 画像と画像の間の余白は gapで指定します。例えば、「gap: 20px;」とすれば良いだけです。

コード

HTML


<div class="grid">
  <div class="grid__item"><img class="grid__img" src="family1.jpg"></div>
  <div class="grid__item"><img class="grid__img" src="family2.jpg"></div>
  <div class="grid__item"><img class="grid__img" src="family3.jpg"></div>
  <div class="grid__item"><img class="grid__img" src="family4.jpg"></div>
  <div class="grid__item"><img class="grid__img" src="family5.jpg"></div>
</div>

CSS


.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
@media screen and (max-width: 767px) { /* スマホ用 */
  .grid {
    grid-template-columns: 1fr;
  }
}
@media screen and (min-width: 768px) and (max-width: 991px) { /* タブレット用 */
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}
