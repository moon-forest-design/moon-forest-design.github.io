Create React App導入手順（Windows10）

  1. Create React Appを導入するには、Node.jsが必要なので、Node.jsをダウンロードしてインストールする。とくに理由がないかぎり LTS版を選択するほうがいい。
  2. Node.jsを利用するためには、ターミナル（コマンドラインツール）が必要になるので、git for windowsなどをダウンロードしてインストールする。Git Bashが使えるようになる。手始めに Node.jsや npmのバージョンを確認する。例：node -v
  3. 適当な場所に Node用の作業フォルダを作成する。例 C:\Users\（ユーザ名）\node\
  4. 上述のフォルダを右クリックし、「Git Bash here」を選択。npxコマンドを使用し Create React AppでReact環境を構築する。例：npx create-react-app react-labo
  5. React環境が構築される時間は、パソコンのスペックによる。スペックによっては 30分以上かかる可能性もある。
  6. 作成したフォルダへ移動。例：cd react-labo
  7. React環境を起ち上げる。例：npm start
  8. ブラウザで「http://localhost:3000/」が表示されれば OK。

どこで何をどうするのか？

  • 作業場所は srcフォルダ内。基礎となるのが index.js。Reactライブラリなどを importし、App.jsを呼び出してレンダーしている。さらに他に importしたいライブラリなどがあれば、ここに書く。
  • アプリを制作する上で中心となるファイルが App.js。App.jsを中心にして他のコンポーネントファイルを importするイメージ。
  • index.jsの CSSは、index.css。全体的に共通するスタイルを設定する。
  • App.jsの CSSは、App.css。

import React from 'react';
import ReactDOM from 'react-dom';
import './index.css';
import App from './App';
import reportWebVitals from './reportWebVitals';

ReactDOM.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
  document.getElementById('root')

App.js


import logo from './logo.svg';
import './App.css';

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <img src={logo} className="App-logo" alt="logo" />
        <p>
          Edit <code>src/App.js</code> and save to reload.
        </p>
        <a
          className="App-link"
          href="https://reactjs.org"
          target="_blank"
          rel="noopener noreferrer"
        >
          Learn React
        </a>
      </header>
    </div>
  );
}

export default App;
