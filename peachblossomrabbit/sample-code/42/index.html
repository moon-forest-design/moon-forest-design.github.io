コード概要

要点

今回は、スムーズスクロール機能です。スムーズスクロールを導入したい場合、いろいろとやり方があるようです。例えば、jQueryなどを使って自分でコードを書いてもいいですし、scroll-behaviorを使う方法もあります。

自分でコードを書くのはしんどいし、scroll-behaviorはブラウザ対応が十分ではないと考えるのであれば、smooth-scroll.jsという jQueryライブラリを利用してみると良いかもしれません。ポイントは以下ようです。

  • ページ内のジャンプ元とジャンプ先を HTMLに書きます。
  • ダウンロードしても良いですが手間なので、HTMLファイル内で jQueryとsmooth-scroll.jsを読み込みます。（下記サンプルコードでは integrity属性等は割愛してあります）
  • main.js内で、smooth-scroll.jsに関する設定をほどこします。

<!-- ページ内のジャンプ先 -->
<h2 id="summary">コード概要</h2>

<!-- jQueryとsmooth-scroll.jsを読み込む -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/smooth-scroll/16.1.3/smooth-scroll.polyfills.min.js"></script>

jQuery


$(function() {
  var scroll = new SmoothScroll('a[href*="#"]', {
    speed: 500, // スクロール速度：1000pxのスクロールにかかる時間をミリ秒単位で表す数値
    speedAsDuration: true, // すべてのアニメーションにまったく同じ時間（速度に設定した値）をかけたいか？
  });
});
