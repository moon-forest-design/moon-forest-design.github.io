React入門
「inputイベント基礎」

コード概要

要点

  • イベント処理1の続き。今回は input要素を使った入力欄。入力したものがコンソールに表示される。
  • onChange属性を使用。仮引数 eventで受取り、「event.target.value」で内容を表示している。onChangeなので入力のたびに表示される。
  • event.targetは、「イベントを発生させたオブジェクトへの参照」を行うプロパティ。ここでは、event.target.valueとし、input要素で入力された値を参照している。
  • 以下は、同じ仕様の入力欄だが、onChange属性内に処理を書かずに、コールバック関数として外に書いた場合のサンプル。

コード

HTML + React


<!-- onChange属性内に処理を書いた場合のコード -->
<div id="root" class="reactRoot"></div>
<script type="text/babel">
  const InputText = () => {
    return (
      <div>
        <input type="text" defaultValue="" onChange={event => console.log(event.target.value)} />
      </div>
    )
  }
  ReactDOM.render(
    <InputText />,
    document.getElementById('root')
  );
</script>

<!-- onChange属性外に処理を書いた場合のコード -->
<div id="root2" class="reactRoot"></div>
<script type="text/babel">
  const InputText2 = () => {
    // コールバック関数
    const handleInput= (event) => {
      // イベントハンドラを書く
      console.log(event.target.value)
    }
    return (
      <div>
        <input type="text" defaultValue="" onChange={handleInput}/>
      </div>
    )
  }
  ReactDOM.render(
    <InputText2 />,
    document.getElementById('root2')
  );
</script>
