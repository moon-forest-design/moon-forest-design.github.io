画像表示に役立つ max-widthの使い方

コード概要

要点

今回も、CSSの max-widthのお話です。例えば仮に、画面幅が 1280pxのモニタがあるとします。CSSで画像の幅を 640pxに指定した場合、次のようにおさまり良く表示されるのでとくに問題はありません。

しかしこれを、画面幅 360pxのスマホで見ると、ちょっと困った感じになります。画面幅が小さくなると、次のように画像がはみ出してしまうのです。デザイン的な意図があってあえてやっていれば構いませんが、そうでないなら困ります。

このようなときは max-widthを指定しておくと、うまいこと解決できます。画面内でおさまるように表示したければ、「max-width: 100%;」とすれば良いだけです。

つまり、画像に指定した widthの値でおさまる画面幅であればそのまま表示されます。おさまらなければ、%で指定した値で、画面の幅（もしくは親要素の幅）にあわせてうまいこと表示されます。max-widthは、状況に応じて widthを上書きしてくれる役立つプロパティです。

コード

HTML


<div>
  <img class="logoImg" src="img/logo-peachblossomrabbit.png">
</div>

CSS


.logoImg {
  width: 640px;
  height: auto;
  max-width: 100%;
}
