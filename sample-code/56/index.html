動画を画面いっぱいに表示する方法

コード概要

要点

今回は、動画を画面いっぱいに表示する方法について書きます。以前、静止画像（jpeg）を画面いっぱいに表示する方法について書きましたが、今回はその動画版です。ポイントは以下ようです。

  • video要素を用意する。状況に応じて必要な属性を各種設定する。autoplay（自動再生）、loop（繰り返し）、playsinline（インライン再生）、muted（音無し）、poster（動画対応していないブラウザ対策用の静止画）など。
  • video要素に直接 src属性を記述して動画を呼び出しても良いが、一般的には source要素を使って指定する。mp4だけでなく、aviや ogvなどの異なるフォーマットを複数指定しておくとユーザーのコーデック環境に左右されにくくなるメリットがある。
  • widthに 100%、heightに 100vhを指定。画像の元サイズが幅 1920pxなので、max-widthで 1920pxを指定。
  • object-fitに coverを指定。coverを指定することで、レスポンシブ対応となる。スマホやタブレットなど異なる画面サイズで表示してもうまく調整して表示してくれる。
  • video要素内に、念のため「Sorry, your browser doesn't support embedded videos.」と書いても良い。ただ、poster属性で静止画像を指定しておけば、とくに必要ないかもしれない。

コード

HTML


<div class="mainVisual">
  <video class="mainVisual__video" autoplay loop playsinline muted poster="amapola.jpg">
    <source src="amapola.mp4" type="video/mp4">
  </video>
</div>

CSS


.mainVisual__video {
  width: 100%;
  max-width: 1920px;
  height: 100vh;
  object-fit: cover;
}
