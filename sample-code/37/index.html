レスポンシブな一覧表の作り方

コード概要

要点

今回は、レスポンシブな一覧表（table）の作り方です。スマートフォンの場合は、一覧表の下に横スクロールバーが表示される仕様です。ポイントは以下ようです。

  • table要素を入れる親要素（.tableBox）を作成します。スマートフォン以外では、幅100%とし、子要素の tableにも継承させます。
  • メディアクエリで、.tableBoxに max-widthと「overflow-x: scroll;」を指定します。max-width値は、データ内容が読みやすく、且つ一覧表をすべて見るためには横スクロールが必要だとユーザーが気づきやすい値に調整しました。
  • table要素も、.tableBoxのmax-width値と同じ値です。

コード

HTML


<div class="tableBox">
  <table class="table">
    <thead>
      <tr>
        <th>見出し</th><th>見出し</th><th>見出し</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>データ</td><td>データ</td><td>データ</td>
      </tr>
      <tr>
        <td>データ</td><td>データ</td><td>データ</td>
      </tr>
      <tr>
        <td>データ</td><td>データ</td><td>データ</td>
      </tr>
    </tbody>
  </table>
</div>

CSS


.tableBox {
  width: 100%;
}
.table {
  width: inherit;
  border-collapse: collapse;
  text-align: center;
  border-top: 1px solid #302833;
  border-left: 1px solid #302833;
}
.table th,
.table td {
  padding: .5em 1em;
  border-right: 1px solid #302833;
  border-bottom: 1px solid #302833;
}
.table th {
  background-color: #ffccaa;
}
@media screen and (max-width: 767px) { /* スマホ用 */
  .tableBox {
    max-width: 400px;
    overflow-x: scroll;
  }
  .table {
    width: 400px;
  }
}

編集後記

あとがき

  • 理想的な幅が決まっている一覧表（table）等をレスポンシブ対応するのは難しいです。一覧表の構造を変えてしまうのも、データの意図が読み取りにくくなるので好ましくありませんし、一覧できない一覧表とはこれいかにと感じます。どうしたって無理があります。悩ましい。
