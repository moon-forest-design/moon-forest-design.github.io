Font Awesomeの手軽な設定方法

コード概要

要点

今回は、Webアイコンフォント「Font Awesome」の設定方法です。CDNを利用して i要素をHTMLファイルへ張り付ける手軽なやり方です。手順は以下ようです。

  1. HTMLファイルの head要素内に CDNコードを貼り付けておく。CDNコードは、例えば、BootstrapCDNなどで調達する。
  2. Font Awesomeへ行く。
  3. 上部メニュー「Icon」をクリックする。
  4. 使用したい Webアイコンフォントをアルファベットで検索する。（例えば、メールのアイコンなら「mail」と入力する）
  5. 検索結果が表示される。濃く表示されているものが無料で使用できる Webアイコンフォントです。薄く表示されているのは有料（Pro版）の Webアイコンフォントです。
  6. 使用してみたいものが見つかったら、その Webアイコンフォントをクリックする。
  7. Webアイコンフォントのページが表示される。大きく Webアイコンフォントが表示されている少し上にある HTMLコードをクリックしてコピーする。（例：<i class="fas fa-envelope"></i>）
  8. HTMLファイル内の表示したい箇所に、コピーした HTMLコード貼り付ける。
  9. あとは、CSSで好きなように編集すれば OK。サイズや色など、好きなように CSSで変更できるので便利です。

コード

HTML（CDNコード例）


<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@fortawesome/fontawesome-free@5.15.3/css/fontawesome.min.css" integrity="sha384-wESLQ85D6gbsF459vf1CiZ2+rr+CsxRY0RpiF1tLlQpDnAgg6rwdsUF1+Ics2bni" crossorigin="anonymous">

HTML（Webアイコンフォントのコード例）


<i class="fas fa-envelope"></i>

CSS


.fa-envelope {
  font-size: 20rem;
  color: #241c1c;
}

編集後記

あとがき

Font Awesomeは、様々な状況に対応できるように工夫されています。しかしそれゆえに、初めて使うときはどうしたらいいのか迷うことがあります。そこで、一番シンプルで手軽に試すことが出来るであろう方法を書きました。Webアイコンフォントを単独で表示するだけなら、これで事足ります。

しかし、文字列の先頭に Webアイコンフォントを表示したいときは、この方法では CSSの書き方が少し面倒な形になるかもしれません。そのような場合は、Font Awesomeを CSSの疑似要素で使うと便利です。
