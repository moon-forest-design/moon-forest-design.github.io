スクロール時に「ふわっ」と表示させる方法
（ScrollReveal.js版）

コード概要

要点

今回は、ふわっと表示です。ページをスクロールしていくと、各コンテンツが少し下の位置からふわっと表れるようにする方法です。以前、jQueryを利用して自作しましたが、今回は jQueryライブラリのScrollReveal.jsを試してみました。ポイントは以下ようです。

  • 事前に、ふわっと表示したい領域（ここでは .content）を用意しておく。
  • jQuery・ScrollReveal.js・main.jsを読み込む。
  • ScrollReveal.jsに関する設定を main.jsで行う。ふわっと表示したい領域（ここでは '.content'）を指定し、あとは、duration・viewFactor・distanceなどの設定を行えば OKです。

コード

HTML


<!-- ふわっと表示したい領域 -->
<section class="content">
～（コンテンツ）～
</section>

<!-- jQuery・ScrollReveal.js・main.jsを読み込む -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<script src="https://unpkg.com/scrollreveal"></script>
<script src="main.js"></script>

jQuery


$(function() {
  ScrollReveal().reveal('.content', { 
    duration: 2000, // アニメーションの開始から終了までの時間
    viewFactor: 0.3, // どれくらい見えたら実行するか指定（値：0～1）
    distance: '100px', // 下から移動表示する場合の長さ
  });
});
