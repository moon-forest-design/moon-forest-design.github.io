React入門1

コード概要

要点

  • reactを試すために CDNを利用。CDNは、開発者用とプロダクション用が用意されている。開発者用を開発するときに指定するものと解釈してしまったが勘違いだったようだ。開発者用は、いわゆる「開発者が中身を見たい」ときに使うためのもの。ゆえに圧縮・最適化されていない。CDNを利用する場合に指定するのはプロダクション用。react.production.min.js、react-dom.production.min.jsを使う。
  • CDNを利用する場合は、開発者用でもプロダクション用でも babel.min.jsも読み込む必要があるようだ。本来は、事前にコンパイルしておけばいいことなので、使用するのはあまり好ましくない。
  • 空のdiv要素（#root）を作る。この中に script要素を使って、p要素を作成するようなイメージ。仮想DOM。
  • script要素には、type属性で "text/babel"を指定。
  • ReactDOM.render()にて、HTMLを差し込む。第一引数に element（HTML文）。第二引数に container（差し込み先）。
  • 第一引数で指定する要素に class属性を使用したらエラーが出た。react（JSX）では className属性を使用する。
  • script要素の内容を外部呼び出し（main.js）にしてみたがエラーが出た。クロスオリジンエラー。サーバを起てれば大丈夫なのかもしれない。
  • これだけだと何のメリットもない感じだが、基礎の基礎となる。

コード

HTML


<!-- reactの使用に必要なものを読み込む -->
<script src="https://unpkg.com/react@17/umd/react.development.js"></script>
<script src="https://unpkg.com/react-dom@17/umd/react-dom.development.js"></script>
<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>

<!-- reactを使用したコード -->
<div id="root" class="reactRoot"></div>
<script type="text/babel">
  ReactDOM.render(
    <p>Reactテスト：はじめの一歩</p>,
    document.getElementById('root')
  );
</script>
