Google Fontsの使い方

コード概要

要点

今回は、Google Fontsを使う方法について書きます。

  1. Google Fontsへアクセスする。
  2. フォントを選択する。（今回は、Tanukizamuraiさんの「Yusei Magic」を試してみます）
  3. 「+ select this style」をクリックする。
  4. 「To embed a font, copy the code into the <head> of your html 」の下に記載されているコードをコピーし、HTMLファイルへ貼り付ける。
  5. 「CSS rules to specify families 」の下に記載されているコードをコピーし、CSSファイルへ貼り付ける。例えば、h1要素に適用したければ、h1要素に対して指定する。
  6. 表示確認。問題がなければ次のように選択したフォントで表示される。

おかしなはがきが、ある土曜日の夕がた、一郎のうちにきました。

かねた一郎さま　九月十九日

あなたは、ごきげんよろしいほで、けっこです。

あした、めんどなさいばんしますから、おいで

んなさい。とびどぐもたないでくなさい。

山ねこ　拝

コード

HTML


  <link rel="preconnect" href="https://fonts.gstatic.com">
  <link href="https://fonts.googleapis.com/css2?family=Yusei+Magic&display=swap" rel="stylesheet">

CSS


h1 {
  font-family: 'Yusei Magic', sans-serif;
}
