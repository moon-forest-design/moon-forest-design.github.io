ハンバーガーボタンのクリックアニメ

コード概要

要点

今回は、ハンバーガーボタンのクリックアニメーションです。ボタンをクリックすると、横三本線がバツ印に変わります。ポイントは以下ようです。

  • 横三本線は span要素で作成します。それを親要素div（.burgerMenu）で囲います。
  • 横三本線には、共通するクラス名（burgerMenu__line）と、それぞれを判別するためのクラス名（burgerMenu__line--1など）を用意しました。
  • .burgerMenu__lineで absoluteしたいので、親要素div（.burgerMenu）に「position: relative;」します。
  • 横三本線は、「width: 100%;」、height: 2px;」、「background-color: #000;」で描いています。
  • それぞれの線（.burgerMenu__line--1、.burgerMenu__line--2、.burgerMenu__line--3）は、absoluteで配置します。それぞれ「top: 6px;」、「top: 14px;」、「top: 22px;」の位置です。
  • 親要素div（.burgerMenu）がクリックされたら、jQueryでクラス名「open」を付加します。付加された状態でクリックされたら、逆に削除します。
  • 親要素div（.burgerMenu）にクラス名「open」が付加されたら、.burgerMenu__line--1と .burgerMenu__line--3は、位置を変えながら回転してバツ印に変形し、.burgerMenu__line--2は透過します。
  • 親要素div（.burgerMenu）にクラス名「open」がすでに付加された状態であれば、三本線に戻ります。

コード

HTML


<div class="burgerMenu">
  <span class="burgerMenu__line burgerMenu__line--1"></span>
  <span class="burgerMenu__line burgerMenu__line--2"></span>
  <span class="burgerMenu__line burgerMenu__line--3"></span>
</div>

CSS


.burgerMenu {
  width: 30px;
  height: 30px;
  position: relative;
}
.burgerMenu__line {
  width: 100%;
  height: 2px;
  background-color: #000;
  position: absolute;
  cursor: pointer;
  transition: .6s;
}
.burgerMenu__line--1 {
  top: 6px;
}
.burgerMenu__line--2 {
  top: 14px;
}
.burgerMenu__line--3 {
  top: 22px;
}
.burgerMenu.open .burgerMenu__line--1 {
  top: 14px;
  transform: rotate(315deg);
}
.burgerMenu.open .burgerMenu__line--2 {
  opacity: 0;
}
.burgerMenu.open .burgerMenu__line--3 {
  top: 14px;
  transform: rotate(-315deg);
}
