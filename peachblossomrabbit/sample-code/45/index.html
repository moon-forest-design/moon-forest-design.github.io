リンク下線をふわっと表示する
ホバーアニメ

コード概要

要点

今回は、リンク下線を表示するホバーアニメーションです。文字列の上にマウスポインターをのせる（ホバーする）と、リンクであることを示す下線が下の方からふわっと表れる仕様です。ポイントは以下ようです。

  • a要素に疑似要素（::after）を作成し、この疑似要素にホバーアニメをほどこします。
  • ホバーされない状態では、非表示（opacity: 0;）にしてあります。
  • 「position: absolute;」を使って、位置を少し下げておきます。（bottom: -8px;）
  • ホバーされたら、表示（opacity: 1;）し、位置を戻します。（bottom: 0;）
  • アニメーションの時間は、0.4sです。（transition: .4s;）
ホバーアニメ「マウスポインターをのせると下線を表示します」

コード

HTML


<div class="hoverAnime">
  <a class="hoverAnime__link" href="">ホバーアニメ「マウスポインターをのせると下線を表示します」</a>
</div>

CSS


.hoverAnime__link {
  display: inline-block;
  position: relative;
}
.hoverAnime__link::after {
  width: 100%;
  height: 2px;
  position: absolute;
  bottom: -8px;
  left: 0;
  content: "";
  border-bottom: 2px solid #302833;
  opacity: 0;
  transition: .4s;
}
.hoverAnime__link:hover::after {
  opacity: 1;
  bottom: 0;
}

編集後記

あとがき

  • 実際に使う場合は、リンクだと分かるような場所で使用することになるでしょうか。例えば、ナビゲーションメニューなどですね。文章内で使用したらうるさくなりそうです。
  • borderの指定（border-bottom: 2px solid #302833;）を、:hoverのほうにつけると変な感じになるので気をつけましょう。
