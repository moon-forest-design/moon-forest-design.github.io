React入門
「関数によるWikipediaのAPI」

コード概要

要点

  • Class Componentで書かれている「React入門「クラス構文によるWikipediaのAPI」」のコードを、Function Componentで書き直した。
  • 気をつけたいポイントは 2つ。
    • stateの初期化は、「React.useState([])」のように配列として行う。0や空文字ではダメ。
    • ライフサイクルメソッド componentDidMount()として React.useEffect()を使うので、React.useEffect()の第2引数に「[]」をつけるのを忘れてはいけない。初回だけで良い。[data]などと書いてしまうと、データ取得し続けてしまうようだ。

コード

HTML + React


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  const App = () => {
    const [data, setData] = React.useState([]);
    React.useEffect(() => {
      const url = "https://en.wikipedia.org/w/api.php?action=opensearch&search=Japan&format=json&origin=*&limit=1";
      fetch(url)
        .then(result => result.json())
        .then(result => setData(result));
    }, []);

    const result = data.map((entry, index) => {
      console.log(entry);
      return <li key={index}>{entry}</li>;
    });
    return <div className="container"><ul>{result}</ul></div>;

  }
  ReactDOM.render(
    <App />,
    document.getElementById("root")
  );
</script>
