コード概要

要点

今回は、スムーズスクロール機能です。仕様は以下ようです。

  • 事前に、HTMLファイル内の各 h2見出しに、それぞれ idをつけておきます。上部メニューには、各 h2見出しへページ内移動するリンクをつけてあります。
  • jQueryにて、a要素（href属性の値に#が付いているもの）がクリックされたら、ページ内移動がスムーズに行われます。

jQuery


$(function() {
	$('a[href^="#"]').click(function() { // a要素（href属性の値に#が付いているもの）をクリックしたら実行
		const $href = $(this).attr("href"); // a要素のhref属性の中身を取得
		const $adjust = $href === "#" ? 0 : 10; // 移動先がピッタリにならないよう調整する　トップ以外は上に 10px余白
		const $target = $($href === "#" || $href === "" ? "html" : $href); // トップへ戻る場合は"html"を代入　他はaリンクのhrefの中身を代入
		const $position = $target.offset().top - $adjust; // 移動先の要素のtop位置を取得　及び$adjustで調整
		$('body, html').animate({
			scrollTop: $position
		}, 500, 'swing'); // animate()を使ってスムーズにページ内移動　スピード500、動作swing
		return false;
	});
});

編集後記

あとがき

  • トップへ戻るボタンを押したときに、いい感じになるので、スムーズスクロール機能は入れておきたいところです。
  • 「scroll-behavior: smooth;」は、ブラウザ対応がもう少し進んでからでしょうか。
