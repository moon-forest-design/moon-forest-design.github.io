画像を遅延読み込みする lazysizesの使い方

コード概要

要点

今回は、画像を遅延読み込みする lazysizesの使い方です。loading属性は未対応のブラウザがあるので、その対策のためのJavaScriptライブラリです。ポイントは以下ようです。

  • 画像を遅延読み込みしたい img要素の src属性を data-srcと書き換える。
  • W3Cのバリデーションでエラーが発生するので、src属性にはダミーを入れておく。
  • img要素の class名に「lazyload」を付与する。

コード

HTML


<ul class="lazyDemo">
  <li class="lazyDemo__item">
    <img class="lazyDemo__img"
      src="kids1.jpg"
      width="640" height="429">
  </li>
  <li class="lazyDemo__item">
    <img class="lazyDemo__img lazyload"
      data-src="kids2.jpg"
      width="640" height="427"
      src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==">
  </li>
  <li class="lazyDemo__item">
    <img class="lazyDemo__img lazyload"
      data-src="kids3.jpg"
      width="640" height="479"
      src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==">
  </li>
  <li class="lazyDemo__item">
    <img class="lazyDemo__img lazyload"
      data-src="kids4.jpg"
      width="640" height="426"
      src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==">
  </li>
  <li class="lazyDemo__item">
    <img class="lazyDemo__img lazyload"
      data-src="kids5.jpg"
      width="640" height="360"
      src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==">
  </li>
</ul>

CSS


.lazyDemo {
  text-align: center;
  list-style-type: none;
}
