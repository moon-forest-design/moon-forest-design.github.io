backgroundで画像を画面いっぱいに表示する方法

コード概要

要点

今回は、CSSの backgroundを使って、画像を画面いっぱいに表示する方法について書きます。いわゆるヒーローイメージを指定する方法です。ポイントは以下ようです。

  • 空の div要素を用意し、CSSの background-imageで背景画像（ヒーローイメージ）を指定する。
  • 高さ（height）を指定してあげないと表示されないので、必ず高さも指定します。画面の高さいっぱいに表示するため 100vhを指定します。
  • background-positionに center、background-sizeに coverを指定することで、画像の位置と大きさをちょうど良く調整します。

コード

HTML


<div class="mainVisual">
  <!-- CSSのbackgroundで画像を背景表示する領域 -->
</div>

CSS


.mainVisual {
  height: 100vh;
  background-image: url("mainVisual.jpg");
  background-position: center;
  background-size: cover;
}

編集後記

あとがき

  • ヒーローイメージは、そのサイトがどのようなサイトなのかをイメージづけるための画像です。ゆえに、とても大切です。エンドユーザーにとって、ファーストビューとなることもあるので、どのようなことを伝えたいのか？　その意図が明確な画像でなければいけません。「あら、きれいな人ね。まあこれでいっか」などという選び方をしてはいけないのです。
  • 指定内容が分かり良いように分けて書きましたが、「background: center/cover url("mainVisual.jpg");」のようにひとまとめにして書くこともできます。
  • background-position、及び background-sizeの指定値によっては、画像のトリミングもできます。
  • backgroundを使って画像を画面いっぱいに表示する方法は、以前から使用されているおなじみのやり方です。しかし、あまり使われなくなってきているかもしれません。今どきは object-fitを使うことが増えてきているようです。
