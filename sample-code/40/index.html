borderを使った三角リストマーカーの作り方

コード概要

要点

今回は、borderを使った三角リストマーカーの作り方です。ポイントは以下ようです。

  • なぜ borderで三角形を作ることができるのかと言えば、width・heightの指定なく、borderに一定の太さをもたせると次のようになるためです。分かり良いように、各borderには色をつけてあります。top（赤）、right（緑）、bottom（黄）、left（青）です。
  • この各borderが接している箇所が斜めになる borderの特性がとくに重要なポイントとなります。あとは、ほしい三角形は 1つだけですから、他の3つを消してあげれば良いわけです。

top（赤）、right（緑）、bottom（黄）の三角形を、borderの透過機能を利用して見えなくしました。これをリストマーカーで使うと、次のようになります。位置は「position: absolute;」で調整してあげれば済みます。三角リストマーカーの完成です。

  • 項目1
  • 項目2
  • 項目3

コード

HTML


<ul class="triangleList">
  <li>項目1</li>
  <li>項目2</li>
  <li>項目3</li>
</ul>

CSS


.triangleList {
  list-style-type: none;
}
.triangleList li {
  position: relative;
  padding-left: 18px;
}
.triangleList li:before {
  display: block;
  content: "";
  position: absolute;
  top: 5px;
  left: 0;
  border-top: 7px solid transparent;
  border-right: 7px solid transparent;
  border-bottom: 7px solid transparent;
  border-left: 7px solid #1e50a2;
}
