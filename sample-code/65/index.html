React入門2

コード概要

要点

  • 前回の React入門1の続き。「コンポーネント」と呼ばれる書き方。
  • ReactDOM.render()の第一引数に直接 HTML要素を書くのではなく、関数のような形で書く。このような書き方をすることで再利用できるようにもなる。
  • 関数名は、一文字目を大文字にしなければいけない。例えば、サンプルコードで見かける<App />もそう。試しに<TextView />としてみた。これは JSXのルールらしい。
  • 昨今は、関数で書くが、少し前までは ES6の classで定義していたそうだ。いくつか書き方があるので、古いコードを見たときに混乱することがあるかもしれない。
  • React.StrictModeは、おそらく JavaScriptの "use strict";みたいなものだろう。

コード

HTML


<div id="root" class="reactRoot"></div>
<script type="text/babel">
  function App() {
    return <p>Reactテスト：コンポーネントを使う</p>;
  }
  ReactDOM.render(
    <React.StrictMode>
      <App />
    </React.StrictMode>,
    document.getElementById('root')
  );
</script>
