コード概要

要点

以下、「関数を理解すればクロージャは難しくない！｜もっこりJavaScript｜ANALOGIC（アナロジック）」を読んでメモ書き。

  • クロージャとは？
    • 関数と、その関数の定義時のコンテキストをセットにした特殊なオブジェクトのこと。
    • 変数 cntの状態を保持し続けているところに注目。
  • クロージャを理解するポイント。
    • 関数は定義時のコンテキストで実行される。（※重要）
    • 関数は第一級オブジェクトなので、定義時のコンテキストとは異なるコンテキスト上でも実行できる。
    • 関数は定義時のコンテキストとは異なるコンテキスト上にある変数に代入された時にクロージャになる。
  • クロージャの用途。
    • グローバル変数を使わずに関数に「状態」を持たせる。（疑似プライベート変数）
    • 関数を作る関数。

コード

JavaScript


var createCounter = function () {
  var cnt = 0;
  return function () {
    cnt += 1;
    console.log(cnt);
  };
};
var counter = createCounter();
counter(); // 1
counter(); // 2
counter(); // 3
